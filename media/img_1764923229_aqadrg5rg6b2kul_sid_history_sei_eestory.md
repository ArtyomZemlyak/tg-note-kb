## Описание изображения

Изображение демонстрирует процесс использования семантических идентификаторов (SIDs) в генеративных рекомендательных системах. Оно показывает:

- Историю пользователя в виде SIDs (SID history)
- Процесс предсказания следующего айтема с использованием LLM
- Ограниченный beam search (Constrained BeamSearch) для генерации рекомендаций
- Полный процесс выравнивания SIDs (Full-Process SID Alignment)

Этот процесс является ключевым компонентом фреймворков, таких как MiniOneRec и OneRec, где айтемы кодируются короткими последовательностями токенов вместо использования миллионов ID, что позволяет использовать архитектуры трансформеров с огромными каталогами айтемов.
