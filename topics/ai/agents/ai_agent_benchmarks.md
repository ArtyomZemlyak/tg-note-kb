# Бенчмарки для оценки агентов ИИ

## Описание

Бенчмарки для агентов ИИ представляют собой стандартизированные наборы задач и метрик, используемые для оценки способностей агентов искусственного интеллекта. Они позволяют сравнивать различные подходы к разработке агентов и измерять прогресс в этой области.

## Основные бенчмарки

### AgenceBench
- Бенчмарк, разработанный для оценки способностей агентов ИИ в реальных сценариях
- Особое внимание уделяется использованию инструментов и планированию
- Использовался для измерения улучшений от подхода LIMI (с 45% до 73%)
- Включает задачи из доменов Vibe Coding и Research Workflows
- Подробности в статье: [[ai/agents/agencebench.md]]

### Terminal-Bench
- Фреймворк для оценки агентов ИИ через выполнение задач в терминальных/консольных средах
- Предоставляет execution harness для создания сред выполнения задач
- Работает в режиме SWE-Bench задач: агенту ставится задача, он решает её, решение проверяется юнит-тестами
- Подробности в статье: [[ai/agents/terminal_bench_framework.md]]

### Vending-Bench 2
- Расширенный бенчмарк для оценки способности ИИ-агентов поддерживать согласованность и эффективность в течение очень длительных периодов (месяцы и годы)
- Моделирует управление вендинговым автоматом в течение года с реалистичными бизнес-сложностями (недоброжелательные поставщики, задержки поставок, возвраты от клиентов)
- Включает режим "Арены" с конкурентным взаимодействием между моделями
- Подробности в статье: [[ai/agents/vending_bench_2_long_term_business_automation_benchmark.md]]

### MLE-bench
- Бенчмарк, состоящий из 71-75 отобранных соревнований Kaggle для оценки способностей инженерии машинного обучения ИИ-агентов
- Ориентирован на полные ML-воркфлоу: от предобработки данных до обучения и оптимизации моделей
- Используется для анализа влияния разнообразия идей на результативность агентов
- Подробности в статье: [[ai/agents/mle_bench.md]]

## Ключевые аспекты оценки

### 1. Использование инструментов
- Способность агента находить и использовать подходящие инструменты
- Эффективность в интеграции различных инструментов в рабочий процесс
- Понимание ограничений каждого инструмента

### 2. Планирование
- Способность создавать эффективные планы решения задач
- Умение адаптировать план при возникновении препятствий
- Оценка сложности и ресурсоемкости задач

### 3. Решение задач
- Эффективность в решении сложных, многоступенчатых задач
- Способность справляться с неожиданными проблемами
- Качество конечного результата

## Значение для разработки агентов

- Бенчмарки предоставляют объективную метрику для сравнения различных подходов
- Помогают исследователям понимать реальный прогресс в области агентного ИИ
- Подчеркивают важность способности агентов использовать инструменты для решения задач
