# Доступные инструменты для агента в базе знаний

## Описание

Этот файл описывает инструменты, которые доступны мне как автономному агенту для работы с базой знаний. На основе анализа структуры и содержимого базы знаний, я могу использовать различные инструменты для поиска, чтения, записи и управления информацией.

## Основные категории инструментов

### 1. Инструменты поиска и анализа

#### search_file_content
- Поиск по содержимому файлов с использованием регулярных выражений
- Позволяет находить конкретные паттерны в файлах базы знаний
- Используется для многоэтапной стратегии поиска информации

#### glob
- Поиск файлов по шаблонам имен (например, "*gpt*.md", "*нейронн*")
- Используется на первом этапе многоэтапной стратегии поиска
- Быстрый способ найти потенциально релевантные файлы по их именам

#### vector_search
- Семантический векторный поиск по всей базе знаний
- Находит семантически похожий контент на основе естественного языка
- Используется на втором этапе многоэтапной стратегии поиска

### 2. Инструменты чтения файлов

#### read_file
- Чтение содержимого отдельных файлов
- Может читать конкретные диапазоны строк в больших файлах
- Основной инструмент для получения детального содержимого файлов

#### read_many_files
- Чтение содержимого нескольких файлов одновременно
- Полезен для анализа нескольких связанных файлов или получения обзора темы
- Поддерживает шаблоны glob для группового чтения файлов

### 3. Инструменты записи и редактирования

#### write_file
- Создание новых файлов в файловой системе
- Используется для добавления новой информации в базу знаний
- Требуется указать абсолютный путь и содержимое файла

#### edit
- Редактирование существующих файлов путем замены текста
- Требует предоставления контекста вокруг изменений
- Используется для обновления существующих знаний

### 4. Инструменты управления файловой системой

#### list_directory
- Получение списка файлов и подкаталогов в указанной директории
- Полезен для понимания структуры базы знаний
- Может игнорировать определенные шаблоны файлов

#### run_shell_command
- Выполнение команд оболочки для выполнения дополнительных задач
- Может запускать скрипты и другие утилиты, если это необходимо
- Поддерживает как фоновые, так и обычные процессы

### 5. Инструменты управления задачами и памятью

#### todo_write
- Создание и управление списком задач для текущего сеанса
- Помогает отслеживать прогресс и организовывать сложные задачи
- Позволяет пользователю понимать прогресс выполнения задачи

#### save_memory
- Сохранение конкретной информации или факта в долговременную память
- Используется для запоминания предпочтений пользователя или важной информации
- Поддерживает как глобальную, так и привязанную к проекту память

#### store_memory
- Хранение информации в памяти для последующего извлечения
- Организация по категориям и тегам
- Поддержка метаданных для более точного поиска

#### retrieve_memory
- Извлечение информации из памяти
- Поддерживает поиск по запросам, категориям и тегам
- Полезно для восстановления ранее сохраненной информации

## Многоэтапная стратегия поиска

Как агент, я использую 3-этапную стратегию поиска информации в базе знаний:

### Этап 1: Файловый поиск
- Извлечение ключевых терминов из запроса
- Использование инструмента `glob` для поиска файлов по шаблонам имен
- Быстрое получение списка потенциально релевантных файлов

### Этап 2: Семантический векторный поиск
- Использование инструмента `vector_search` для семантического поиска
- Нахождение семантически похожего контента по всей базе знаний
- Получение топ 5-10 наиболее релевантных документов

### Этап 3: Уточняющий поиск по содержимому
- Использование инструмента `search_file_content` для поиска конкретных терминов
- Чтение наиболее релевантных файлов с помощью `read_file` и `read_many_files`
- Глубокий анализ содержимого для получения полной информации

## Применение инструментов в работе

В рамках моей задачи по анализу и структурированию информации я использую:

- **Файловые поисковые инструменты** для понимания структуры базы знаний
- **Инструменты чтения** для изучения существующего контента
- **Инструменты записи и редактирования** для добавления новой информации
- **Инструменты поиска** для многоэтапного анализа
- **Инструменты управления задачами** для организованной работы
