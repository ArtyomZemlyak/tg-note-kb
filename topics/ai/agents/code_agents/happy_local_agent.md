# Happy - Локальный агент для интеграции с Claude Code

## Описание

Happy - это мобильное и веб-приложение для Claude Code и Codex, которое позволяет использовать Claude Code с любого устройства с энд-то-энд шифрованием. Особенностью проекта является автоматическая инъекция контекста без явных вызовов Claude, что позволяет создать более глубокую интеграцию с рабочим циклом разработки.

## Основные возможности

- **Мобильный доступ к Claude Code** - Проверка прогресса агента программирования с телефона
- **Пуш-уведомления** - Оповещения о необходимости разрешений или ошибках
- **Мгновенное переключение между устройствами** - Возможность передачи контроля с одного устройства на другое одним нажатием
- **Энд-то-энд шифрование** - Ваши данные никогда не передаются в незашифрованном виде
- **Автоматическая инъекция контекста** - Локальный агент может подкидывать нужный контекст без явного вызова Claude
- **Интеграция с CLI** - Использование команды `happy` вместо `claude` для запуска агента

## Техническая реализация

Happy работает как обертка (wrapper) для Claude Code. При запуске команды `happy` вместо `claude` запускается AI-агент через обертку. Когда необходимо управлять агентом с мобильного устройства, сессия перезапускается в удаленном режиме. Для возврата к локальному управлению достаточно нажать любую клавишу.

Ключевая особенность - интерактивный режим, в котором локальный агент может автоматически распознавать запросы пользователя и подкидывать соответствующий контекст без необходимости явного вызова Claude.

## Архитектура

### Компоненты проекта

- **happy-cli** - Командный интерфейс для Claude Code и Codex
- **happy-server** - Серверная часть для зашифрованной синхронизации
- **happy-coder** - Мобильный клиент (исходный проект)

### Безопасность

- Сквозное шифрование данных
- Открытый исходный код (MIT License)
- Без телеметрии и отслеживания

## Применение

Happy особенно полезен для:

- Инженеров, проверяющих прогресс своих AI-агентов во время создания проектов
- Разработчиков, которые хотят управлять процессами программирования с мобильных устройств
- Пользователей, требующих высокого уровня безопасности для работы с кодом
- Сценариев, где требуется глубокая интеграция с рабочим процессом Claude Code

## Новые концепции и термины

- **Локальный агент** - Процесс, который работает локально и может автоматически предоставлять контекст Claude Code
- **Интерактивный режим** - Режим работы, при котором локальный агент отслеживает запросы пользователя и автоматически предоставляет контекст
- **Инъекция контекста** - Процесс автоматического предоставления релевантного контекста Claude Code на основе запроса пользователя
- **Обертка для Claude Code** - Инструмент, который расширяет возможности Claude Code, добавляя новые функции

## Примеры применения

- Автоматическое предоставление релевантного контекста при обнаружении запроса пользователя
- Удаленный мониторинг выполнения задач Claude Code
- Переключение между устройствами во время работы с агентом
- Интеграция с мобильными устройствами для управления процессом программирования
