# sgr-core: Фреймворк для агентов с использованием Schema-Guided Reasoning (SGR)

## Краткое описание

sgr-core (также известный как SGR-Deep-Research) - это фреймворк с открытым исходным кодом для создания интеллектуальных агентов исследований, использующих Schema-Guided Reasoning (SGR). Фреймворк предоставляет структурированный подход к рассуждению с использованием схем, что позволяет создавать более предсказуемые и точные ИИ-агенты.

## Основная информация

### Цель и применение

sgr-core предназначен для:
- Создания интеллектуальных агентов исследования с использованием SGR
- Обеспечения структурированного подхода к рассуждению ИИ-агентов
- Повышения предсказуемости и точности в вызовах инструментов
- Поддержки частных исследований с использованием локальных моделей

### Архитектура

- **Расширяемая архитектура**: Предоставляет интерфейс BaseAgent, который можно расширять
- **Двухфазная архитектура**: Реализует двухфазный подход к рассуждению
- **SGR (Schema-Guided Reasoning)**: Использует структурированные схемы для управления процессом рассуждения ИИ-агентов
- **Совместимость с OpenAI API**: Предоставляет совместимый с OpenAI REST API

## Новые концепции и термины

- **Schema-Guided Reasoning (SGR)**: Подход, при котором рассуждения ИИ-агента направляются структурированными схемами
- **sgr-core**: Ядро фреймворка SGR-Deep-Research
- **Hybrid SGR+Tools**: Гибридный подход, объединяющий SGR с нативными вызовами инструментов
- **BaseAgent**: Базовый интерфейс агента, который можно расширять

## Связь с проектом Vampi

sgr-core разрабатывается в рамках организации vamplabAI и имеет отношение к экосистеме проекта Vampi - терминального агента, созданного на основе небольших моделей в режиме PlanAct. sgr-core может использоваться как часть более широкой экосистемы Vampi для продвинутых исследований и разработки ИИ.

## Примеры применения

- Агенты для исследования и анализа данных
- Структурированное автоматизированное рассуждение
- Автоматизация рутинных задач программирования
- Поддержка принятия решений на основе данных
