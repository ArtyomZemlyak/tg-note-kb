# Релиз sgr-core 0.3.0

## Краткое описание

Стабильный релиз sgr-core 0.3.0, включающий важные исправления, новые функции и улучшения архитектуры фреймворка.

## Основная информация

### Изменения в версии 0.3.0

- Исправлено поведение фазы выбора действия агента вызова инструментов (_select_action_phase) - #45 (contributor: virrius)
- Добавлен простой пример CLI клиента для начала работы и тестирования - #32 (contributor: kv-gits)
- Проведен бенчмарк simpleqa - #43 (contributor: maksimov-m)
- Реализована декомпозиция и перенос инструментов в пакет core.tools - #48 (contributor: EvilFreelancer)
- Исправлена/удалена динамическая дата из промптов - #51 (contributor: vakovalskii)
- Обновлены README и документация по участию в проекте - #49 (contributor: maksimov-m)

### Улучшения и нововведения

- **Декомпозиция инструментов**: Инструменты были декомпозированы и перенесены в отдельный пакет core.tools, что улучшает архитектуру фреймворка и облегчает поддержку
- **Улучшения в логике агента**: Исправлена логика агента, особенно поведение фазы выбора действий, что повышает стабильность и предсказуемость работы
- **Примеры и документация**: Добавлены примеры для новых пользователей и улучшена документация для облегчения начала работы
- **Результаты бенчмарков**: Закреплены результаты бенчмарков для оценки производительности

## Новые концепции и термины

- **core.tools**: Новый пакет, содержащий декомпозированные инструменты
- **_select_action_phase**: Фаза выбора действия в агенте вызова инструментов
- **Simple CLI client**: Простой клиент командной строки для тестирования и начала работы

## Примеры применения

- Бенчмаркинг производительности агентов
- Тестирование новых инструментов и функций
- Быстрое начало работы с фреймворком
- Структурированное использование инструментов в агентах
