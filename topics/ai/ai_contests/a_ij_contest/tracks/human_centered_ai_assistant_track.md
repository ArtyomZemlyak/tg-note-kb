# Трек Human-centered AI Assistant в AI Journey Contest 2025

## Описание трека

Human-centered AI Assistant - это трек соревнования AI Journey Contest 2025, направленный на разработку персонализированного ассистента на основе GigaChat.

## Цель трека

Цель трека - создать персонализированного ассистента, адаптирующегося под индивидуальные потребности пользователя. Ассистент должен учитывать предпочтения, стиль общения и контекст конкретного пользователя для предоставления более релевантных и полезных ответов.

## Особенности трека

- Использование GigaChat в качестве основы для разработки ассистента
- Учет индивидуальных предпочтений и контекста пользователя
- Участникам предоставляются токены для API
- Возможность получить дополнительно 1 млн токенов

## Требования к ассистенту

Ассистент, разрабатываемый в рамках этого трека, должен:
- Адаптироваться к индивидуальным потребностям пользователя
- Использовать контекст прошлых взаимодействий
- Обеспечивать качественное взаимодействие на основе GigaChat
- Демонстрировать понимание пользовательских предпочтений

## Значение трека

Трек важен для развития более персонализированных и удобных ИИ-ассистентов, которые могут лучше понимать и предвосхищать потребности пользователей. Это ключевой аспект для создания действительно полезных ИИ-систем в повседневной жизни и работе.
