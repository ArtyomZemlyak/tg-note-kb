# alphaXiv

## Описание

alphaXiv - проект, созданный для обработки и анализа научных статей по искусственному интеллекту с использованием современных ИИ-инструментов, включая DeepSeek OCR.

## Использование DeepSeek OCR

### Обработка статей

- alphaXiv использовал DeepSeek OCR для обработки более 500 000 статей по ИИ
- Модель использовалась для извлечения данных из таблиц и диаграмм
- Цель - анализ самых популярных бенчмарков и датасетов

### Эффективность

- Обработка 500 000+ статей обошлась всего в $1000
- По сравнению с Mistral OCR, которая до этого считалась одной из лучших по соотношению цена/качество, процесс с ней обошелся бы в $7500
- Экономия по сравнению с Mistral OCR составила $6500
- Это демонстрирует эффективность DeepSeek OCR для массовой обработки документов

## Планируемые функции

### Конвертация PDF в Markdown

- alphaXiv планирует релиз датасета со статьями из arXiv, сразу переведенными из PDF в формат Markdown
- Использование DeepSeek OCR для автоматической конвертации
- Значительное упрощение доступа к содержимому научных публикаций
- Преобразование PDF в Markdown является особенно мощным применением технологии, позволяющим автоматизировать обработку научных публикаций

## Значение

Проект alphaXiv показывает, как современные ИИ-инструменты могут быть использованы для автоматизации обработки научной литературы и создания более доступных датасетов для исследований. Проект демонстрирует значительные улучшения в эффективности обработки документов по сравнению с существующими решениями.
