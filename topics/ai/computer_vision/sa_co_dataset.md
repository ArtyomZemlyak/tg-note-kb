# SA-Co Dataset - Датасет для сегментации с понятиями

## Описание

SA-Co (Segment Anything with Concepts) — это крупнейший датасет для задачи Promptable Concept Segmentation (PCS), созданный специально для обучения и оценки модели SAM 3. Датасет включает в себя миллионы изображений и видео с разнообразными понятиями и их сегментациями, что делает его наиболее полным корпусом для концептуальной сегментации на сегодняшний день.

## Структура датасета

### Основные компоненты
- **SA-Co/HQ**: 5.2M высококачественных изображений с 4M уникальными именными фразами
- **SA-Co/SYN**: 38M именных фраз и 1.4B масок из синтетического датасета
- **SA-Co/EXT**: 15 внешних датасетов с хард-негативами
- **SA-Co/VIDEO**: 52.5K видео с 24.8K уникальными именными фразами

### Онтология датасета
- 22M сущностей, охватывающих 17 основных и 72 подкатегории
- Более чем 50-кратное увеличение количества понятий по сравнению с существующими бенчмарками
- Разнообразие доменов: природные сцены, урбанистические пейзажи, промышленные объекты, медицинские изображения

## Методология создания

### Четырехфазный дата-движок
1. **AI-аннотаторы** предлагают кандидатские именные фразы
2. **AI-верификаторы** (дообученная Llama 3.2) оценивают качество масок
3. **Человеческий контроль** фокусируется на ошибках, удваивая производительность
4. **Качественный контроль** обеспечивает точность и полноту разметки

### Процесс генерации
- Использование комбинации человеческих экспертов, моделей SAM и дообученных LLM для создания разметки
- Стратегия "человек в петле" для обработки сложных случаев
- Автоматизация где это возможно, с человеческим надзором в критических точках

## Стандарты оценки

### SA-Co/Gold
- 7 подмножеств с тройной проверкой разметки человеком
- Используется для оценки качества модели на критических приложениях
- Представляет различные домены: еда, искусство, робототехника, вождение и др.

### SA-Co/Silver
- 10 подмножеств, охватывающих разнообразные домены
- Включает промышленные, медицинские, аэрофотоснимки и другие специализированные области
- Бенчмарки для оценки производительности на специфических сценариях

### SA-Co/VEVal
- 3 подмножества для оценки видео: YT-Temporal-1B, SmartGlasses и SA-V
- Оценка производительности в задачах видео-сегментации и трекинга
- Покрывает сложные сценарии с окклюзиями и изменением освещения

## Технические характеристики

### Размер и масштаб
- Общий объем: ~5.2M изображений и 52.5K видео
- Более 1.4 млрд. масок сегментации
- Более 4M уникальных именных фраз
- Покрытие более чем 22M визуальных сущностей

### Качество данных
- Высокая точность разметки благодаря многоуровневой проверке
- Валидация как по качеству масок, так и по семантической корректности
- Хард-негативы для улучшения способности модели различать похожие объекты

## Значение для компьютерного зрения

### Прорыв в масштабе
- SA-Co значительно превосходит по масштабу предыдущие датасеты сегментации
- Включает более чем в 50 раз больше понятий по сравнению с существующими бенчмарками
- Представляет гораздо более широкую онтологию визуальных понятий

### Поддержка PCS
- Датасет специально разработан для задачи Promptable Concept Segmentation
- Содержит разнообразные именные фразы, которые могут быть использованы как текстовые промпты
- Включает примеры сложных случаев, которые ранее были под задачей для моделей сегментации

### Новые возможности
- Возможность обучать модели для сегментации широкого спектра понятий
- Поддержка zero-shot оценки на редких и необычных объектах
- Возможность тонкой настройки для специфических приложений и доменов

## Вызовы и ограничения

### Этические соображения
- Необходимость проверки размеченных изображений на наличие чувствительного контента
- Обеспечение разнообразия и отсутствия предвзятости в разметке
- Защита частных и персональных данных в публичных субсетах

### Технические ограничения
- Сложность качественной разметки для всех типов объектов
- Проблемы с границами объектов, особенно для перекрывающихся или полупрозрачных
- Вариации в интерпретации визуальных понятий разными аннотаторами
