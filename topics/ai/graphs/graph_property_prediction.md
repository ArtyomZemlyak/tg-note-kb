# Предсказание свойств графа (Graph Property Prediction, GPP)

## Описание

Предсказание свойств графа (GPP) - это класс задач в области графовых нейронных сетей, где цель состоит в предсказании глобальных свойств всего графа или его структурных характеристик. Эти задачи требуют глубокого понимания всей структуры графа и часто связаны с дальнодействующими зависимостями.

## Типы задач GPP

### Структурные свойства

#### Диаметр графа (Diameter)
- **Определение**: максимальное кратчайшее расстояние между любыми двумя узлами в графе
- **Требует**: понимания глобальной структуры графа
- **Сложность**: нужна информация из всех частей графа

#### Эксцентриситет (Eccentricity)
- **Определение**: максимальное расстояние от узла до любого другого узла в графе
- **Требует**: информацию о самых удаленных узлах
- **Сложность**: вычислительно сложная задача

#### Радиус графа (Radius)
- **Определение**: минимальная из максимальных дистанций от узла до всех остальных узлов
- **Требует**: глобального понимания структуры графа
- **Сложность**: требует знания расстояний до всех узлов

### Топологические свойства
- Связность графа
- Плотность графа
- Кластерный коэффициент
- Центральность узлов

## Связь с дальнодействующими зависимостями

Задачи GPP особенно чувствительны к проблеме over-squashing, так как:
- Требуют информации из экспоненциально большого рецептивного поля
- Нужно понимание всей структуры графа
- Распределение информации по большому расстоянию затруднено

## Методы решения

### gLSTM для GPP
- Архитектура gLSTM показывает передовые результаты на задачах GPP
- Способна сохранять и извлекать информацию из удаленных частей графа
- Эффективно обрабатывает задачи, требующие понимания структуры на больших расстояниях

### Другие подходы
- Graph Transformers с механизмом внимания
- K-hop агрегация для лучшего потока информации
- Graph Rewiring для улучшения связности

## Бенчмарки

### Long Range Graph Benchmark (LRGB)
- Стандартизированный набор задач для оценки способности GNN обрабатывать дальнодействующие зависимости
- Включает задачи GPP среди прочих

### Специфические задачи
- Предсказание диаметра
- Предсказание эксцентриситета
- Задачи на основе синтетических графов

## Применение

### Химия и биология
- Предсказание свойств молекул на основе молекулярных графов
- Понимание взаимодействий между удаленными атомами

### Социальные сети
- Определение ключевых узлов в социальных структурах
- Анализ влияния на всю сеть

### Биоинформатика
- Анализ белковых структур
- Предсказание функций генов
