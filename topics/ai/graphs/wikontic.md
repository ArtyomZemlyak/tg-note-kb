# Wikontic: Пайплайн для извлечения графов знаний из текста

## Описание

Wikontic — это современный многоступенчатый пайплайн, разработанный для извлечения графов знаний из текстов, который создает высококачественные, согласованные с онтологией графы знаний из текстов открытой предметной области. Пайплайн достигает передовых результатов в сохранении информации и эффективности использования токенов.

## Основная информация

### Что такое Wikontic

Wikontic — это пайплайн, который превращает текст в компактные и согласованные с онтологией Wikidata графы знаний, без лишнего шума и дубликатов. Графы в дальнейшем можно использовать на downstream задачах вроде QA. Получаемые графы сохраняют высокую долю фактической информации из исходных текстов.

### Архитектура пайплайна

Пайплайн Wikontic состоит из трех основных этапов:

1. **Извлечение кандидатов-триплетов с контекстной метаинформацией**:
   - Извлечение кандидатных триплетов с квалификаторами из текста
   - Сохранение контекстной информации для верификации знаний

2. **Обеспечение ограничений на типы и отношения, основанных на Wikidata**:
   - Применение ограничений, основанных на онтологии Wikidata
   - Фильтрация и корректировка триплетов в соответствии с онтологией

3. **Нормализация сущностей для уменьшения дубликатов**:
   - Снижение дублирования через нормализацию идентификаторов сущностей
   - Объединение различных представлений одной и той же сущности

## Особенности

### Согласованность с онтологией

- **Согласованность с Wikidata**: Пайплайн строго придерживается онтологии Wikidata, что обеспечивает высокую семантическую согласованность
- **Качество графов**: Создает компактные, онтологически-согласованные и хорошо связанные графы знаний
- **Отсутствие дубликатов**: Активно устраняет дублирование информации через нормализацию сущностей

### Эффективность

- **Экономия токенов**: Использует значительно меньше выходных токенов (в 20 раз меньше, чем популярный GraphRAG от Microsoft)
- **Компактное представление**: Генерирует графы с использованием менее чем 1000 выходных токенов
- **Высокая производительность**: Снижение использования токенов примерно в 3 раза по сравнению с AriGraph и менее чем в 1/20 по сравнению с GraphRAG

### Обеспечение структурированного контекста для LLM

- **Верифицируемая основа для LLM**: Обеспечивает структурированную, проверяемую основу для больших языковых моделей
- **Улучшенная точность**: Помогает снизить галлюцинации в LLM за счет использования структурированного представления знаний
- **Независимость от текстового контекста**: Может использоваться как автономное решение без необходимости в текстовом контексте

## Производительность

### Результаты на бенчмарках

- **MINE-1 бенчмарк**: Передовые результаты по сохранению информации (86%)
- **MuSiQue**: 96% правильных сущностей ответов появляются в сгенерированных триплетах
- **HotpotQA**: 76.0 F1-мера
- **MuSiQue**: 59.8 F1-мера (сопоставимо или превосходит базовые линии на основе расширенной генерации)
- **Сравнение с KGGEN**: Оказалось лучше чем метод KGGEN (авторы которого выступают с докладом на NeurIPS) с первого запуска на бенчмарке MINE

### Сравнение с другими методами

- **Превосходит KGGEN**: На бенчмарке MINE оказался лучше чем метод KGGEN с первого запуска
- **Меньше токенов**: В 20 раз меньше выходных токенов по сравнению с GraphRAG от Microsoft
- **Качество на уровне текстовых методов**: Качество на multi-hop датасетах сопоставимо с методами, которые дополнительно используют исходный текст

## Применение

### Downstream задачи

- **Ответы на вопросы (QA)**: Графы знаний могут использоваться для решения сложных QA-задач, опираясь только на граф
- **Многозвеньевые задачи (multi-hop)**: Эффективен на датасетах, требующих логических переходов для ответа
- **Обогащение контекста LLM**: Предоставляет структурированный контекст для больших языковых моделей

### Интеграция с системами ИИ

- **Нейросимволические системы**: Может использоваться как источник структурированных знаний
- **Рекомендательные системы**: Возможность использования в LLM4KGC (Large Language Models for Knowledge Graph Completion)
- **Системы памяти для ИИ-агентов**: Графы знаний могут использоваться как компонент долгосрочной памяти
