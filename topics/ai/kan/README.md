# Сети Колмогорова-Арнольда (KAN)

Этот раздел посвящен сетям Колмогорова-Арнольда (KAN) — новой архитектуре нейронных сетей, представленной как альтернатива традиционным многослойным перцептронам (MLP). KAN основаны на теореме Колмогорова-Арнольда и обеспечивают лучшую интерпретируемость и параметрическую эффективность.

- [[mathematics/kolmogorov_arnold_theorem.md]] - Теорема Колмогорова-Арнольда
- [[architectures/kan_architecture.md]] - Архитектура сетей KAN
- [[basis_functions/basis_functions_overview.md]] - Обзор базисных функций
- [[applications/scientific_computing.md]] - Применения в научных вычислениях
- [[optimization/training_strategies.md]] - Стратегии обучения KAN

## Основные особенности

- **Архитектура**: Обучаемые функции на рёбрах вместо фиксированных активаций
- **Базисные функции**: Ключевое значение выбора базисной функции
- **Применения**: Особенно эффективны в научных вычислениях и решении дифференциальных уравнений
- **Преимущества**: Лучшая интерпретируемость, параметрическая эффективность, меньшее спектральное смещение
