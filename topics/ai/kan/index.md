# Сети Колмогорова-Арнольда (KAN)

## Общее описание

Сети Колмогорова-Арнольда (KAN) представляют собой новую архитектуру нейронных сетей, предложенную как альтернатива традиционным многослойным перцептронам (MLP). В отличие от MLP, которые сначала линейно смешивают входы, а затем применяют фиксированную функцию активации (смешать → активировать), KAN применяют обучаемое одномерное преобразование φ(x) к каждой входной координате на рёбрах сети перед их агрегацией (активировать → смешать).

Архитектура KAN основана на теореме представления Колмогорова-Арнольда (KAT), которая утверждает, что любую непрерывную многомерную функцию можно разложить в суперпозицию непрерывных одномерных функций.

## Математическое описание

Один слой KAN определяется преобразованием:

x_q^(l+1) = Σ_{p=1..P} φ_{q,p}^(l)(x_p^(l))

Ключевое нововведение заключается в параметризации этих одномерных отображений φ(x) как линейной комбинации базисных функций Bₖ(x) с обучаемыми коэффициентами cₖ:

φ(x) = Σ_{k=1..K} cₖ Bₖ(x)

## Преимущества KAN

- **Параметрическая эффективность**: KAN требует O(GW²L) параметров по сравнению со сложностью O(G²W⁴L) для эквивалентного MLP на базе ReLU, где G — размер сетки, W — ширина, а L — глубина.
- **Интерпретируемость**: Сети обладают лучшей способностью к интерпретации своих решений.
- **Меньшее спектральное смещение**: В отличие от MLP, KAN по своей природе менее подвержены спектральному смещению, что важно для физически-информированных нейронных сетей (PINN).
- **Адаптивные базисные функции**: Возможность адаптации базисных функций под конкретную задачу.

## Применение

- Решение дифференциальных уравнений в частных производных (ДУЧП)
- Физически-информированные нейронные сети (PINN)
- Научные вычисления
- Задачи с резкими переходами и разрывами
