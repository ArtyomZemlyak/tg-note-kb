# Методология Google по оценке качества данных для обучения LLM

## Обзор

Google опубликовал отчет, в котором подтверждается расхожее мнение: небольшое количество высококачественных данных для обучения LLM лучше, чем большое количество данных низкого качества. Техника обучения, например, LoRA (Low-Rank Adaptation), основана на этом же предположении.

## Ключевые принципы

### Качество данных превыше количества
- Небольшое количество хорошо подобранных данных может быть более эффективным, чем большие объемы низкокачественных данных
- Это подтверждает важность тщательного отбора и подготовки обучающих датасетов
- Качество данных критически влияет на итоговую производительность модели

### Применение к техникам адаптации
- Методы параметрически эффективного обучения, такие как LoRA, предполагают использование небольших наборов высококачественных данных
- Эти методы демонстрируют, что фокус на качестве, а не на количестве, может дать отличные результаты

## Методология оценки качества данных

### Прямой опрос LLM
- Подход заключается в том, чтобы напрямую спрашивать LLM, является ли данный фрагмент данных хорошим или нет
- Это позволяет автоматизировать предварительную оценку качества данных
- Использование LLM для оценки собственных обучающих данных - инновационный подход к качественной оценке

### Кластеризация и выборка
- После предварительной оценки данные подвергаются кластеризации
- Из кластеров, где данные имеют разные метки качества, выбираются сэмплы для ручной разметки
- Это позволяет более эффективно использовать ресурсы для ручной проверки данных

### Активное обучение
- Подход представляет собой обычное активное обучение (active learning) в один шаг
- Система автоматически определяет, какие данные требуют ручной проверки
- Позволяет сфокусироваться на наиболее проблемных или неоднозначных примерах

## Применение и преимущества

### Повышение эффективности разметки
- Метод позволяет сосредоточить человеческие усилия на самых информативных примерах
- Экономит ресурсы при подготовке высококачественных датасетов
- Улучшает общий процесс оценки и улучшения качества данных
