# A-Vibe и A-Vision от Авито

## Описание

A-Vibe и A-Vision - первые российские языковые модели для электронной коммерции с глубокой оптимизацией под русский язык, разработанные технологической платформой Авито. Обе модели были открыты в открытом доступе и дообучены на данных e-commerce, заточены под задачи маркетплейса.

## Характеристики

### A-Vibe
- Модель для задач, связанных с e-commerce
- Оптимизирована под русский язык
- Занимает первые места в различных рейтингах
- Применяется для анализа документации, создания ИИ-ассистентов и автоматизации контента

### A-Vision
- Мультимодальная модель (предположительно, по названию)
- Часть экосистемы моделей от Авито
- Также оптимизирована под русский язык и задачи e-commerce

## Технические возможности

- **Обработка длинных текстов**: обе модели могут работать с длинными текстами (примерно 100 страниц)
- **Вызов внешних инструментов**: могут вызывать внешние инструменты для выполнения расчетов, поиска данных в базах или обращения к API-сервисам
- **Создание автономных ИИ-агентов**: с помощью этих моделей можно создавать автономных ИИ-агентов

## Применение

- Анализ документации
- Создание ИИ-ассистентов
- Автоматизация контента
- Задачи электронной коммерции
- Работа с e-commerce данными

## Разработка и инвестиции

Компания Авито потратила более полумиллиарда рублей на разработку ИИ-моделей. Команда Авито также русифицировала большой пул популярных тестов для замера качества моделей.

## Доступность

Все инструменты доступны разработчикам бесплатно на сайте Hugging Face.

## Значение для рынка

- Первые российские модели, специализирующиеся на e-commerce
- Глубокая оптимизация под русский язык
- Открытый исходный код для сообщества разработчиков
- Вклад в развитие ИИ-экосистемы в России
