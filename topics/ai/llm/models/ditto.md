# Ditto

## Описание
Ditto - это передовая платформа для генерации высококачественных данных для редактирования видео на основе инструкций. Она объединяет мощь генераторов изображений и видео, создавая уникальный набор данных Ditto-1M с миллионом примеров, что позволяет обучать модели, такие как Editto, с выдающимися результатами.

## Архитектура
- Новаторский подход к генерации данных для редактирования видео
- Интеграция возможностей редактирования изображений с генерацией видео
- Эффективная архитектура модели с пониженной стоимостью и повышенным качеством
- Дистиллированная модель с временным усилителем (temporal enhancer)
- Интеллектуальный агент для фильтрации и контроля качества

## Компоненты
- **Фреймворк Ditto**: Голистическая система для решения задач инструкционного редактирования видео
- **Модель Editto**: Новая передовая модель для редактирования видео на основе инструкций
- **Датасет Ditto-1M**: Одно миллион примеров редактирования видео высокого качества
- **Интеллектуальный агент**: Создает разнообразные инструкции и строго отфильтровывает результаты

## Датасет Ditto-1M
- Содержит один миллион примеров редактирования видео высокой достоверности
- Создан с использованием более чем 12,000 GPU-часов вычислений
- Построен с помощью масштабируемого конвейера данных Ditto
- Доступен через датасет Hugging Face
- Включает CSV-файлы для обучения модели с помощью DiffSynth-Studio
- Включает метаданные JSON для симуляции реальных условий

## Модель Editto
- Новая передовая модель для редактирования видео на основе инструкций
- Обучена на датасете Ditto-1M с использованием стратегии обучения по программе (curriculum learning)
- Демонстрирует превосходную способность следовать инструкциям
- Устанавливает новый стандарт (SOTA) в области инструкционного редактирования видео

## Обучение и использование
- Поддерживает обучение с использованием CSV-файлов из датасета Hugging Face
- Поддерживает многоузловое обучение через DiffSynth-Studio
- Может использоваться с ComfyUI с квантованными и дистиллированными моделями
- Требует меньше вычислительных ресурсов (832×480×73 видео требуют 11ГБ видеопамяти и ~4 мин на A6000)
