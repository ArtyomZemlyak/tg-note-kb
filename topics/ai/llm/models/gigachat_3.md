# GigaChat 3 - Новое поколение моделей от Sber AI

## Общее описание

GigaChat 3 - это новое поколение больших языковых моделей от Sber AI, представленное в ноябре 2024 года. Линейка включает две основные модели: GigaChat 3 Ultra Preview - флагманскую модель с 702 миллиардами параметров, и GigaChat 3 Lightning - компактную модель для локального и высоконагруженного использования.

## Модели линейки GigaChat 3

### GigaChat 3 Ultra Preview
- **Общие параметры**: 702 миллиарда
- **Активные параметры**: 36 миллиардов
- **Архитектура**: Mixture-of-Experts (MoE) с 256 экспертами
- **Количество слоев**: 64
- **Количество голов внимания**: 64
- **Особенности**: Флагманная instruct-модель, спроектированная для сложных задач и высококачественной генерации

### GigaChat 3 Lightning
- **Общие параметры**: 10 миллиардов
- **Активные параметры**: 1.8 миллиарда
- **Архитектура**: Mixture-of-Experts (MoE) 
- **Особенности**: Компактная MoE-модель для локального и высоконагруженного использования

## Архитектурные особенности

### Mixture-of-Experts (MoE)
Обе модели GigaChat 3 используют архитектуру Mixture-of-Experts, которая позволяет:
- Активировать только подмножество параметров во время инференса
- Сохранять высокое качество при меньших вычислительных требованиях
- Использовать разреженные вычисления для повышения эффективности

### Multi-head Latent Attention (MLA)
Архитектура моделей включает в себя Multi-head Latent Attention - подход к экономии памяти, при котором тензоры ключей и значений сжимаются в пространство меньшей размерности перед сохранением в KV-кеш, что улучшает производительность моделирования по сравнению с традиционным GQA (Grouped Query Attention).

## Тренировочный процесс

### Предварительная подготовка (Pretrain)
- **Объем данных**: 14 триллионов токенов
- **Процесс**: Интенсивное обучение на обширном корпусе текстов

### Выравнивание модели (Alignment)
- **Этап 1.5**: Промежуточный этап между предварительным обучением и финальной настройкой
- **Chain-of-Thought RL**: Использование обучения с подкреплением для цепочечного мышления
- **Supervised Finetuning**: Контролируемая донастройка для специфических задач

### Основные характеристики
- **Скорость инференса**: Модели оптимизированы для ускорения на MTP (Matrix Tensor Processor)
- **Функции**: Интерпретатор кода, функция поиска, система пользовательской памяти
- **Сравнительная эффективность**: GigaChat 3 Ultra показывает конкурентоспособные результаты на различных бенчмарках, включая MMLU, MMLU-RU, HumanEval+ и другие

## Применение

GigaChat 3 используется в различных ИИ-приложениях:
- Высококачественная генерация текста
- Системы понимания и анализа естественного языка
- Инструменты кодирования и анализа
- Поисковая функциональность
- Пользовательские системы памяти

## Особенности

- Высокое качество обработки и генерации русскоязычного текста
- Поддержка различных форматов взаимодействия
- Интеграция с API для разработчиков
- Совместимость с современными фреймворками для работы с ИИ
