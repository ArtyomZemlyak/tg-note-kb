# Обработка видео в Qwen3-Omni

## Общий обзор

Qwen3-Omni включает в себя улучшенные возможности обработки видео, которые позволяют модели эффективно обрабатывать видеоданные с динамическим соотношением кадров и поддержкой потоковой обработки.

## Особенности видеообработки

### Динамическая подача видео

- Основное изменение для видеомодальности заключается в том, как видео подаётся на вход модели
- Теперь изображения и аудио чередуются не фиксированными двухсекундными блоками, как раньше, а динамически - в потоке
- Используется гибкое соотношение кадров, что делает мультимодальный стриминг более естественным

### Сравнение с предыдущими версиями

- В предыдущих версиях использовались фиксированные двухсекундные блоки для чередования изображений и аудио
- Новый подход позволяет модели более гибко адаптироваться к структуре видео

## Потоковая обработка

### Мультимодальный стриминг

- Динамическое чередование позволяет создавать более естественный мультимодальный стриминг
- Поддерживается обработка длинных видео (до 40 минут)
- Используется асинхронный prefilling для эффективности

## Интеграция с другими модальностями

### Совместная обработка

- Видеообработка интегрирована с текстовой и аудио обработкой
- Используются общие механизмы для обработки разных модальностей
- Сохраняется согласованность между различными типами данных

## Performance

- Поддерживает SOTA результаты в визуальных задачах
- Сохраняет качество при обработке сложных видеосценариев
