# Прунинг в LLM

## Краткое описание

Этот раздел посвящен различным методам прунинга (обрезки) в больших языковых моделях (LLM). Прунинг - это техника оптимизации, при которой удаляются несущественные параметры или структуры из модели для уменьшения размера модели, снижения вычислительных затрат и улучшения скорости инференса при сохранении качества.
