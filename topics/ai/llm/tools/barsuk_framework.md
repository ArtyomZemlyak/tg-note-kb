# Фреймворк "Барсук" от Яндекса

## Описание

Фреймворк "Барсук" - внутренний инструмент Яндекса, разработанный командой ПромоСтраниц для автоматизации процесса создания контента с использованием больших языковых моделей (LLM). Создан примерно в 2023 году как часть нейроредакции.

## Основные функции

- Содержит промпты для каждого этапа работы с контентом (от брифа до финальной версии)
- Автоматизирует рутинные задачи по подготовке промптов
- Интегрирован с нейросетями для генерации текста
- Включает систему макросов для упрощения процесса

## Архитектура

### Промпты по этапам
- Бриф: преобразование требований клиента в структурированный запрос
- План: создание структуры будущей статьи
- Генерация: создание текста на основе промптов
- Постобработка: очистка и доработка текста

### Интеграция с автоматизацией
- Использование эмулятора действий пользователя
- Автоматическое формирование чатов с нейросетью
- Выгрузка диалогов в формат, удобный для редакторов

## Рабочий процесс

1. **Ввод данных**: бриф клиента и тема статьи
2. **Автоматическая генерация промптов** с использованием "Барсука"
3. **Отправка запросов** в нейросеть
4. **Сбор и обработка ответов**
5. **Формирование результата** в формате, готовом для редактирования

## Влияние на производительность

- Сокращение времени на создание текста с 8 часов до 1-2 часов
- Увеличение объема контента в 2 раза при том же бюджете
- Повышение качества текстов за счет структурированного подхода

## Применение

Фреймворк используется в нейроредакции Яндекса для:
- Создания лонгридов для клиентов
- Автоматизации рутинных операций
- Унификации процесса работы с LLM
- Повышения качества и скорости генерации контента
