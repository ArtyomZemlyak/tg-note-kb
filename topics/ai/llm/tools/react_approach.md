# ReAct: Синергия рассуждений и действий в языковых моделях

## Краткое описание

ReAct (Reasoning and Acting) - это подход, представленный в научной работе "React: Synergizing reasoning and acting in language models", который объединяет рассуждение и действие в больших языковых моделях (LLM). Метод позволяет моделям генерировать рассуждения для планирования и выполнения действий, создавая цепочки "мышлений" (thoughts), которые помогают в решении сложных задач.

## Основная информация

### Основные концепции

1. **Цепочки рассуждений**: Генерация промежуточных рассуждений (thoughts), которые помогают модели принять решение о следующем действии.

2. **Интеграция действий**: Возможность выполнения внешних действий (например, вызовов инструментов, поиска в интернете) на основе рассуждений модели.

3. **Итеративный процесс**: Цикл рассуждений → действия → наблюдения, повторяющийся до достижения цели.

### Архитектура и методология

ReAct включает в себя:

- **Текстовые рассуждения**: Генерация "мыслей" модели о текущей ситуации и следующих шагах
- **Активации инструментов**: Возможность вызова внешних инструментов на основе рассуждений
- **Обратная связь**: Интеграция результатов действий в последующие рассуждения

### Преимущества

- Повышенная интерпретируемость процесса принятия решений
- Лучшая адаптация к сложным задачам, требующим нескольких шагов
- Эффективное использование внешних инструментов
- Лучшее понимание контекста через явные рассуждения

## Новые концепции и термины

- **Thought (Мысль)**: Промежуточное рассуждение модели, объясняющее следующее действие
- **Action (Действие)**: Вызов внешнего инструмента или выполнение какого-либо действия
- **Observation (Наблюдение)**: Результат действия, возвращаемый модели для дальнейших рассуждений
- **ReAct цепочка**: Последовательность рассуждений и действий вида Thought → Action → Observation

## Примеры применения

- Ответы на вопросы, требующие поиска информации
- Выполнение сложных многошаговых задач
- Взаимодействие с API и внешними системами
- Решение логических задач
