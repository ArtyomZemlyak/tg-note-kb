# SLiME: Фреймворк пост-тренировки LLM для масштабирования обучения с подкреплением

## Описание

SLiME (Structured Language Model Editor) - это фреймворк пост-тренировки больших языковых моделей (LLM) для масштабирования обучения с подкреплением (RL). Он обеспечивает две основные возможности:

1. **Высокопроизводительная тренировка**: Поддерживает эффективное обучение в различных режимах за счет интеграции Megatron с SGLang
2. **Гибкая генерация данных**: Позволяет создавать произвольные рабочие процессы генерации обучающих данных через кастомные интерфейсы генерации данных и серверные движки

SLiME является основой для обучения RL моделей GLM-4.5 и GLM-4.6 и поддерживает различные модели, включая серии Qwen3, DeepSeek V3 и Llama 3.

## Особенности

- Высокопроизводительная тренировка за счет интеграции Megatron с SGLang
- Гибкие рабочие процессы генерации данных
- Кастомные интерфейсы генерации данных
- Серверные движки для генерации данных
- Поддержка различных семейств моделей (Qwen, DeepSeek, Llama)
- Возможности двухэтапного обучения

## Архитектура

Архитектура состоит из следующих компонентов:

- **Тренировка (Megatron)**: Отвечает за основной процесс обучения, считывает данные из буфера данных и синхронизирует параметры с модулем вывода (rollout)
- **Вывод (Rollout) (SGLang + маршрутизатор)**: Генерирует новые данные (включая награды/результаты верификаторов) и сохраняет их в буфер данных
- **Буфер данных**: Модуль-мост, управляющий инициализацией промптов, кастомными данными и методами генерации вывода

## Связь с SGLang

SLiME имеет глубокую интеграцию с SGLang:

- SLiME объединяет Megatron с SGLang для высокопроизводительной тренировки
- Модуль вывода использует SGLang + маршрутизатор для генерации новых данных
- Аргументы SGLang поддерживаются в SLiME с префиксом `--sglang-`
- Архитектура показывает SGLang как основной компонент на этапе вывода

## Использование

SLiME поддерживает различные типы аргументов:

1. Аргументы Megatron (передаются напрямую)
2. Аргументы SGLang (с префиксом `--sglang-`)
3. Специфические аргументы SLiME (описаны в `slime/utils/arguments.py`)
