# PP-DocLayoutV2

## Описание

PP-DocLayoutV2 - это продвинутая модель анализа макета документов, разработанная как часть фреймворка PaddlePaddle. Модель отвечает за понимание структуры документа и определение порядка чтения элементов страницы. Это ключевой компонент первого этапа в двухэтапной архитектуре PaddleOCR-VL.

## Архитектура

PP-DocLayoutV2 состоит из двух последовательно соединенных сетей:

### 1. RT-DETR детекционная модель
- Основана на архитектуре DETR (Detection Transformer)
- Отвечает за эффективное обнаружение и классификацию элементов макета
- Обнаруживает текстовые блоки, таблицы, формулы, графики и другие структурные элементы страницы

### 2. Pointer Networks
- Легковесная сеть с шестью трансформерными слоями
- Используется для предсказания правильного порядка чтения элементов
- Обеспечивает понимание человеческого порядка чтения на странице

## Функции

### Обнаружение элементов макета
- Текстовые блоки
- Таблицы
- Формулы
- Графики
- Изображения
- Другие структурные элементы

### Определение порядка чтения
- Использует Pointer Networks для предсказания человеческого порядка чтения
- Учитывает логическую структуру документа
- Обрабатывает сложные макеты с несколькими колонками

## Преимущества

### Высокая точность
- Точное обнаружение элементов макета благодаря RT-DETR
- Правильное определение порядка чтения с использованием Pointer Networks

### Стабильность
- Устойчив к сложным многостолбцовым макетам
- Устойчив к смешанным элементам (текст, изображения, таблицы)

### Интеграция
- Тесно интегрирована с вторым этапом PaddleOCR-VL-0.9B
- Подготавливает чистый список элементов для точного распознавания

## Применение

PP-DocLayoutV2 используется как первый этап в двухэтапной системе PaddleOCR-VL, где она:
1. Анализирует структуру страницы документа
2. Обнаруживает и классифицирует элементы макета
3. Предсказывает порядок чтения
4. Создает структурированное представление документа для дальнейшего распознавания
