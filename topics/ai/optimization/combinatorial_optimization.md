# Комбинаторная оптимизация

## Общее описание

Комбинаторная оптимизация - это раздел оптимизации, в которой переменные принимают дискретные значения или значения из конечного множества. Эта область занимается задачами нахождения оптимального объекта из конечного или счетного бесконечного множества. Комбинаторные задачи часто формулируются как задачи оптимизации на графах, сетях, матроидах и других дискретных структурах.

## Исторические этапы развития

### 1. Ранние этапы (1930-1950-е годы)
- **Проблема коммивояжёра**: одна из первых формально сформулированных комбинаторных задач
- **Методы решения**: простые эвристики и переборные методы
- **Практические приложения**: задачи планирования и логистики

### 2. Развитие теории (1950-1970-е годы)
- **Алгоритм ветвей и границ**: развитие систематических методов решения
- **Линейное программирование**: развитие методов, которые позже были адаптированы для комбинаторных задач
- **Теория графов**: развитие теоретических основ для многих комбинаторных задач

### 3. Современные методы (1970-2000-е годы)
- **Метаэвристики**: развитие генетических алгоритмов, имитации отжига, муравьиных алгоритмов
- **Приближенные алгоритмы**: развитие методов с доказанными гарантиями качества
- **Целочисленное программирование**: развитие точных методов решения

### 4. Современные подходы (2000-е годы - настоящее время)
- **Гибридные методы**: сочетание точных и приближенных методов
- **Машинное обучение в оптимизации**: применение ML для улучшения алгоритмов
- **AI-эволюционные методы**: использование LLM и агентных систем, как в FunSearch и DeepEvolve

## Основные классы задач

### 1. Задачи упаковки и покрытия
- **Проблема упаковки в корзину**: упаковка объектов различных размеров в минимальное количество контейнеров
- **Проблема покрытия множеств**: выбор минимального набора подмножеств для покрытия всех элементов
- **Проблема вершинного покрытия**: нахождение минимального набора вершин, инцидентных всем рёбрам

### 2. Задачи маршрутизации
- **Проблема коммивояжёра (TSP)**: нахождение кратчайшего пути, проходящего через все города
- **Проблема маршрутизации транспортных средств (VRP)**: планирование маршрутов для флота транспортных средств
- **Проблема китайского почтальона**: нахождение кратчайшего пути, проходящего по каждому ребру хотя бы раз

### 3. Задачи на графах
- **Максимальный независимый набор**: нахождение наибольшего набора вершин без рёбер между ними
- **Максимальный клик**: нахождение наибольшего полного подграфа
- **Раскраска графов**: присвоение минимального числа цветов вершинам так, чтобы соседние вершины имели разные цвета

### 4. Планирование и распределение
- **Задача о назначениях**: оптимизация распределения ресурсов
- **Задачи планирования**: оптимизация расписаний с различными ограничениями
- **Проблема распределения потоков**: оптимизация потоков в сетях

## Методы решения

### 1. Точные методы
- **Перебор**: перебор всех возможных решений (практически применим только к очень маленьким задачам)
- **Метод ветвей и границ**: систематический перебор с отсечением заведомо неоптимальных ветвей
- **Целочисленное программирование**: формулировка задач как целочисленных программ и решение с помощью специализированного ПО

### 2. Приближенные алгоритмы
- **Приближенные алгоритмы**: алгоритмы с доказанными гарантиями качества
- **Алгоритмы с постоянным фактором приближения**: обеспечивают решения, отличающиеся от оптимального не более чем на фиксированный множитель
- **Полностью полиномиальные приближенные схемы (FPTAS)**: семейства приближенных алгоритмов с произвольной точностью

### 3. Эвристические методы
- **Генетические алгоритмы**: эволюционные подходы к поиску решений
- **Имитация отжига**: метод, вдохновлённый физическим процессом отжига
- **Муравьиные алгоритмы**: методы, вдохновлённые поведением муравьёв
- **Алгоритмы локального поиска**: улучшение решений через последовательность локальных изменений

### 4. Современные AI-методы
- **Обучение с подкреплением**: использование RL для обучения политикам решения задач
- **Нейронные комбинаторные оптимизаторы**: применение нейронных сетей к задачам оптимизации
- **LLM-эволюционные методы**: использование LLM в эволюционных процессах, как в системах FunSearch и DeepEvolve

## Приложения и примеры

### 1. Промышленные приложения
- **Оптимизация цепочек поставок**: минимизация затрат на транспортировку и хранение
- **Планирование производства**: оптимизация расписаний и распределения ресурсов
- **Дизайн сетей**: оптимизация топологий телекоммуникационных и транспортных сетей

### 2. Вычислительные приложения
- **Оптимизация компиляторов**: распределение регистров, планирование инструкций
- **Планирование задач в распределенных системах**: эффективное распределение вычислительных задач
- **Оптимизация архитектуры нейронных сетей**: поиск оптимальных топологий и гиперпараметров

### 3. Научные приложения
- **Биоинформатика**: выравнивание последовательностей, предсказание структур белков
- **Физика**: оптимизация конфигураций систем частиц
- **Химия**: оптимизация молекулярных структур и реакционных путей

## Современные вызовы и тенденции

### 1. Масштабирование задач
- **Большие размеры задач**: необходимость решения всё более крупных задач
- **Распределённые вычисления**: использование кластеров и облака для решения задач
- **Алгоритмы, работающие с ограничениями памяти**: эффективные алгоритмы для работы с большими данными

### 2. Учет неопределенности
- **Робастная оптимизация**: методы, устойчивые к неопределенности в данных
- **Стохастическая оптимизация**: учет вероятностных моделей в оптимизации
- **Адаптивные алгоритмы**: алгоритмы, адаптирующиеся к изменяющимся условиям

### 3. Интеграция с ИИ
- **Использование LLM для генерации решений**: как в системах FunSearch и DeepEvolve
- **Гибридные системы**: объединение традиционных методов с методами ИИ
- **Обучение на ходу**: адаптация алгоритмов на основе предыдущего опыта
