# Математические оптимизационные задачи в системах AlphaEvolve и FunSearch

## Общее описание

Математические оптимизационные задачи представляют собой класс задач, в которых требуется найти наилучшее решение из возможного множества решений. В контексте систем AlphaEvolve и FunSearch, эти задачи служат тестовыми площадками для автоматического открытия алгоритмов и новых решений сложных проблем.

## Задачи, исследованные с помощью систем ИИ

### Задача 6.2: Константа оптимизации для неотрицательных функций

Пусть C_5 обозначает наибольшую константу, для которой выполняется определенное неравенство для всех неотрицательных функций f: R → R. Задача заключается в определении значения этой константы.

#### Значение:
- Эта задача представляет собой типичный пример в области гармонического анализа
- Системы ИИ используются для поиска конструкций, которые могут улучшить оценки этой константы
- Результаты включают автоматически сгенерированные конструкции и их автоконволюции

### Задача 6.59: Подмножества сетки без равнобедренных треугольников

Для натурального числа n, пусть C_59(n) обозначает размер наибольшего подмножества [n] = {1, ..., n}, которое не содержит (возможно, плоских) равнобедренных треугольников.

#### Значение:
- Комбинаторная задача, связанная с избеганием определенных конфигураций
- ИИ системы ищут верхние и нижние оценки для C_59(n)
- Подходит для автоматического открытия, поскольку требует перебора сложных комбинаторных структур

### Задача о упаковке (Packing problems)

Различные задачи упаковки, такие как:
- Упаковка единичных гексагонов в правильный гексагон
- Упаковка единичных кубов в больший куб
- Упаковка в дилатации (задача 6.35)

#### Значение:
- Прикладные задачи с важными применениями в геометрии и инженерии
- Требуют геометрической интуиции и точного анализа
- Хорошо подходят для эволюционных методов, поскольку позволяют использовать генеративные подходы

### Задача о kissing numbers (числах поцелуев)

Задача определения наименьшего размера подмножества в n-мерном пространстве, которое не содержит изосцелических треугольников (равнобедренных треугольников).

#### Значение:
- Классическая проблема в комбинаторной геометрии
- Связана с вопросами о плотности упаковки сфер
- Представляет интерес для теоретической и прикладной математики

## Методы решения с помощью ИИ

### Подход AlphaEvolve
- Использует эволюционные методы для поиска оптимальных конструкций
- Может работать с более длинными программами и более сложными структурами
- Поддерживает более длительные вычисления и использование ускорителей

### Подход FunSearch
- Использует программирование с подсказками
- Эволюционирует отдельные функции, а не целые программы
- Более быстрая оценка, но может быть ограничена в масштабе задачи

## Результаты и достижения

### Эксперименты с различными размерами LLM
- Эксперименты показали, что более дорогие, большие LLM требуют меньше вызовов для достижения хороших результатов по сравнению с более дешевыми, маленькими LLM
- Это указывает на важность масштаба модели для эффективности оптимизации

### Улучшения, найденные системами
- AlphaEvolve улучшила известные результаты в нескольких задачах, включая проблему чисел поцелуев
- Системы нашли новые конструкции для задач упаковки, превзошедшие существующие решения

## Сравнение эффективности

### Влияние вычислительных ресурсов
- Использование большего количества параллельных потоков в AlphaEvolve позволяет быстрее находить хорошие конструкции, но с большими общими вычислительными затратами
- Баланс между скоростью открытия и общими затратами критически важен для планирования экспериментов

## Значение для математики

Применение систем ИИ к математическим оптимизационным задачам:

1. **Ускорение открытий**: Автоматизированный поиск позволяет быстрее находить новые решения
2. **Новые подходы**: ИИ может находить решения, неожиданные для человека-математика
3. **Экспериментальная математика**: Позволяет тестировать гипотезы и получать эвристики
4. **Генерация гипотез**: Автоматическое формирование гипотез на основе найденных паттернов
