# Модели эмбеддингов для RAG-систем

## Описание

Модели эмбеддингов преобразуют текст в векторные представления, которые используются для поиска релевантных документов в RAG-системах. Качество эмбеддингов напрямую влияет на точность извлечения информации.

## Модели для русского языка

### ai-forever/FRIDA

- Российская модель для генерации эмбеддингов
- Оптимизирована для русскоязычного контента
- Хорошая семантическая точность для русского языка

### BAAI/bge-m3

- Мультиязычная модель от BAAI
- Поддерживает 100+ языков, включая русский
- Высокая универсальность и точность

### intfloat/multilingual-e5-large

- Модель для мультиязычного эмбеддинга
- Основана на архитектуре E5
- Подходит для задач, требующих понимания нескольких языков

### Qwen3-Embedding-8B

- Модель от Alibaba для генерации эмбеддингов
- Часть фреймворка Qwen3
- Высокая производительность и точность

## Методы оценки качества эмбеддингов

- Высокая семантическая точность
- Способность к корректному пониманию контекста
- Устойчивость к шуму и вариациям формулировок

## Практические рекомендации

### Для русскоязычных приложений

- ai-forever/FRIDA для специфичного русскоязычного контента
- BAAI/bge-m3 для мультиязычных систем
- Qwen3-Embedding-8B для интеграции с Qwen-моделями

### Общие рекомендации

- Оценка моделей на задачах, близких к вашему домену
- Учет вычислительных ресурсов и времени инференса
- Тестирование нескольких моделей для сравнения
