# PARO - Pattern-Aware Learning

## Краткое описание

PARO (Pattern-Aware Learning) - это методика в области искусственного интеллекта, которая фокусируется на обучении моделей явным паттернам рассуждения, а не на объеме или безупречности рационалей (объяснений). Подход основан на исследовании «Reasoning Pattern Matters: Learning to Reason without Human Rationales».

## Основная информация

PARO представляет собой новую эпоху в обучении моделей рассуждению, где ключ к качеству решения задач заключается не в количестве человеческих рационалей, а в явном усвоении шаблона рассуждения. Это особенно эффективно для задач с устойчивым процедурным шаблоном решения.

### Ключевые особенности

- **Фокус на шаблоне рассуждения**: Не объём и не безупречность рационалей, а явное усвоение шаблона рассуждения является ключом к качеству
- **Экономичность**: Обучение на 10× меньшем количестве человеческих рационалей с минимальной потерей качества
- **Автоматическое генерирование рационалей**: PARO может автоматически генерировать рационали, следуя заранее заданному шаблону
- **Устойчивость к "испорченным" рационалям**: Даже частично "испорченные" рационали с сохранением шаблона почти не теряют в качестве

### Методология

PARO использует подход SFT+RLVR:
- **SFT (Supervised Fine-Tuning)**: На первом этапе обучает модель выдавать явные траектории рассуждения (рационали)
- **RLVR (Reinforcement Learning with Verifiable Rewards)**: На втором этапе усиливает стратегию по проверяемым ответам без "золотых" рационалей

## Новые концепции и термины

- **Patterned reasoning (шаблонное рассуждение)**: Тип задач, где решение следует постоянному шаблону для всех экземпляров задачи
- **Adaptive reasoning (адаптивное рассуждение)**: Тип задач, где стратегия решения меняется от случая к случаю (олимпиадная математика, сложное планирование)
- **RFTD (Rollout-based Forking Token Detection)**: Метод анализа поведения модели, определяющий, какие токены в цепочке рассуждений являются настоящими "узловыми" решениями
- **РАционали**: Явные объяснения или цепочки рассуждений, генерируемые моделью

## Примеры применения

PARO особенно эффективен для задач, где решение следует постоянному шаблону:

1. **Классификация по фиксированным правилам**
2. **Верификация по чек-листу**
3. **Извлечение информации по схеме**
4. **Финансовый анализ и классификация**
