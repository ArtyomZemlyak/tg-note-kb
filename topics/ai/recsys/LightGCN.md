# LightGCN: Упрощение и усиление графовой свертки для рекомендаций

## Описание

LightGCN (Light Graph Convolutional Network) - это упрощенная версия NGCF (Neural Graph Collaborative Filtering), разработанная для эффективного обучения в задачах рекомендаций. Модель исключает ненужные компоненты, сохраняя при этом основную эффективность графовых сверток.

## Основные принципы

LightGCN был разработан на основе анализа NGCF и выявления избыточных компонентов:
- MLP-агрегаторы в NGCF могут приводить к переобучению
- Нелинейности могут мешать эффективному обучению
- Простое усреднение соседей может быть более эффективным

## Архитектура

### Упрощенная агрегация

В отличие от NGCF, LightGCN использует простое усреднение:
- Устранены MLP-агрегаторы
- Устранены нелинейности (.ReLU)
- Только линейная комбинация соседских эмбеддингов

### Light Graph Convolution (LGC)

Операция свертки в LightGCN:
```
h_u^{(l+1)} = Σ_{v∈N(u)} (1/√|N(u)||N(v)|) * h_v^{(l)}
```

Где:
- `h_u^{(l)}` - эмбеддинг пользователя/айтема u на слое l
- `N(u)` - множество соседей узла u
- `|N(u)|` - степень узла u

### Финальное представление

- Финальный эмбеддинг вычисляется как сумма эмбеддингов со всех слоев
- `h_u = Σ_{l=0}^L α_l * h_u^{(l)}`
- α_l - обучаемые веса слоев

## Преимущества

- **Простота**: минимальное количество параметров
- **Эффективность**: быстрое обучение и инференс
- **Отсутствие переобучения**: благодаря упрощению архитектуры
- **Отсутствие перепараметризации**: только начальные эмбеддинги являются обучаемыми параметрами
- **Превосходные результаты**: часто превосходит NGCF и другие сложные модели

## Отличия от NGCF

| Аспект | NGCF | LightGCN |
|--------|------|----------|
| Агрегация | MLP + ReLU | Простое усреднение |
| Нелинейности | Да | Нет |
| Обучаемые параметры | MLP-агрегаторы + эмбеддинги | Только эмбеддинги |
| Сложность | Высокая | Низкая |
| Склонность к переобучению | Да | Нет |

## Предотвращение over-smoothing

- Использование суммы со всех слоев вместо только последнего
- Если использовать только последний слой, возникает проблема over-smoothing
- Комбинирование слоев позволяет сохранить информацию на разных уровнях

## Эмпирические результаты

LightGCN показывает стабильно лучшие результаты по сравнению с:
- NGCF
- GCMC
- GCMC+
- GC-MC
- И другими GNN-моделями
