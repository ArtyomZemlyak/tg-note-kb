# DAS: Dual-Aligned Semantic IDs Empowered Industrial Recommender System

## Описание

DAS (Dual-Aligned Semantic IDs) - это промышленная рекомендательная система, которая использует семантические идентификаторы, обучаемые с учетом коллаборативного сигнала. Основная идея заключается в том, что во время обучения семантических ID авторы напрямую внедряют коллаборативный сигнал, что способствует лучшему взаимодействию между семантическими и коллаборативными аспектами рекомендательной системы.

## Ключевые особенности

- **Семантические ID с коллаборативным сигналом**: В отличие от традиционных подходов, где семантические ID обучаются независимо, DAS обучает их с учетом коллаборативного поведения пользователей
- **Двойное выравнивание**: Метод выравнивает как айтемы, так и пользователей на семантическом уровне, что позволяет более точно моделировать предпочтения
- **Применение пользовательских ID**: В дополнение к айтемным ID, метод также включает пользовательские семантические ID в рекомендательную систему

## Архитектура

DAS расширяет традиционные подходы к обучению семантических представлений следующими способами:

1. Внедрение коллаборативного сигнала прямо в процесс обучения семантических ID
2. Использование двойного выравнивания для согласованности как айтемов, так и пользователей
3. Интеграция пользовательских семантических ID в общую архитектуру рекомендаций

## Преимущества

- Улучшенная персонализация за счет более глубокой интеграции семантической и коллаборативной информации
- Более точное моделирование пользовательских предпочтений через выравнивание на семантическом уровне
- Возможность работы с разреженными данными за счет двойного подхода
