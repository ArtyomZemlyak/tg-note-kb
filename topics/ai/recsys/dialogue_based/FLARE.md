# FLARE - Гибридная архитектура для рекомендаций

## Описание

FLARE - гибридная система рекомендаций, предложенная в рамках проекта REGEN. Эта архитектура сочетает традиционные методы коллаборативной фильтрации с современными языковыми моделями для генерации контекстуальных объяснений рекомендаций.

## Архитектура

FLARE состоит из двух основных компонентов:

1. **Классическая модель рекомендаций** - используется архитектура, аналогичная SASRec, для предсказания следующего товара, который может заинтересовать пользователя.

2. **Легковесная языковая модель** - используется Gemma 2B для генерации нарративов на основе предсказания, сделанного первой моделью.

## Принцип работы

FLARE работает по последовательному принципу:
- Первая модель (SASRec-подобная) предсказывает следующий товар на основе истории взаимодействий пользователя
- Затем легковесная языковая модель (Gemma 2B) генерирует контекстуальное объяснение для предложенного товара

## Преимущества

- Позволяет улучшить метрики за счет включения пользовательской обратной связи
- Менее вычислительно затратна по сравнению с едиными LLM-архитектурами
- Сохраняет преимущества как коллаборативной фильтрации, так и генерации текста

## Связь с REGEN

FLARE является одной из двух архитектур, протестированных в рамках проекта REGEN. Эксперименты показали, что даже последовательное использование FLARE (предсказание следующего элемента + языковая модель) улучшает метрики по сравнению с традиционными подходами.

## Сравнение с LUMEN

- FLARE: гибридный подход с разделением задач между разными моделями
- LUMEN: единая архитектура на базе LLM для всех задач
