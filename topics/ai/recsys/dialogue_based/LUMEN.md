# LUMEN - Единая LLM-архитектура для рекомендаций

## Описание

LUMEN - единая архитектура на основе больших языковых моделей, предложенная в рамках проекта REGEN. Это первая модель, заявленная именно в статье REGEN, как архитектура, способная совмещать обработку user-item интеракций и текстовых взаимодействий в одной модели.

## Архитектура

LUMEN представляет собой единую большую языковую модель (на базе Gemma LLM), которая обучается для совместного выполнения нескольких задач:

1. **Обработка критики** - понимание пользовательской обратной связи и критики в текстовой форме
2. **Генерация рекомендации** - предсказание следующего товара, который может заинтересовать пользователя
3. **Создание нарратива** - генерация контекстуальных объяснений для рекомендаций

## Особенности

- **Единый процесс обучения** - все задачи решаются в рамках одной модели
- **End-to-end обучение** - модель учится "решать", когда выдать ID товара, а когда продолжить генерировать текст
- **Учет интеракций** - обработка и user-item взаимодействий, и текстовых взаимодействий
- **Gemma LLM в основе** - использование конкретной архитектуры языковой модели

## Принцип работы

LUMEN обучена end-to-end решать задачу: на вход подается история покупок и, опционально, текстовая критика, после чего модель должна порекомендовать следующий товар и сгенерировать о нем контекстуальный нарратив.

Модель учится "решать", когда выдать ID товара, а когда продолжить генерировать текст, что позволяет ей адаптивно реагировать на потребности диалога.

## Преимущества

- **Единая архитектура** - более согласованные и связные выводы
- **Лучшее качество согласованности** - по сравнению с гибридными подходами
- **Контекстная генерация** - интеграция рекомендаций и текста в одном процессе
- **Гибкость** - возможность реагировать на сложные текстовые запросы

## Связь с REGEN

LUMEN является одной из двух архитектур, протестированных в рамках проекта REGEN. Как единая модель на основе end-to-end подхода, LUMEN показала лучшее качество согласованности по сравнению с гибридной системой FLARE.

## Сравнение с FLARE

- FLARE: гибридный подход с разделением задач между разными моделями
- LUMEN: единая архитектура на базе LLM для всех задач
