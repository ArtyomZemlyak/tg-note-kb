# RecGPT (Alibaba)

## Описание

RecGPT - это технический подход от Alibaba, описанный в техническом отчете, представляющий собой использование больших языковых моделей для рекомендательных систем. Хотя не упомянуты конкретные результаты A/B тестирования, система представляет собой важный этап в развитии LLM-based рекомендательных подходов.

## Архитектура

### Основные идеи

1. **Общая концепция**:
   - Использование предобученных LLM для задач рекомендаций
   - Преобразование традиционных рекомендательных задач в формат, подходящий для языковых моделей
   - Адаптация архитектуры LLM для обработки пользовательских профилей и айтемов

2. **Интеграция с существующими системами**:
   - Возможность интеграции с существующими рекомендательными пайплайнами
   - Использование LLM как улучшенный компонент в существующей архитектуре

## Особенности

- Описан в виде технического отчета от Alibaba
- Представляет собой раннюю реализацию подхода LLM для рекомендаций
- Показывает потенциал использования LLM в рекомендательных системах до появления более зрелых решений от Google и Kuaishou
