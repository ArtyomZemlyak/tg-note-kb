# TIGER (Google/YouTube)

## Описание

TIGER (Generative Recommendations) - это предшествующая работа, которая задала направление для последующего развития в области генеративных рекомендаций, включая фреймворк PLUM. Работа была представлена теми же авторами, включая Ed Chi и Lichan Hong, до выхода более масштабной статьи о PLUM на конференции RecSys.

## Архитектура

### Основные компоненты

1. **Семантические идентификаторы (SIDs)**: В оригинальной работе TIGER SIDs формировались через RQ-VAE поверх текстового описания товара. Эксперименты проводились на открытых датасетах Amazon.

2. **RQ-VAE (Residual Quantization Variational Autoencoder)**: Метод, используемый для создания семантических представлений айтемов. В TIGER подход основывался преимущественно на текстовом описании айтемов, без использования коллаборативного сигнала или мультимодальных эмбеддингов.

3. **Токенизация айтемов**: Основная идея заключалась в преобразовании айтемов в семантические идентификаторы, которые затем могут быть использованы в языковых моделях. Это позволяет обрабатывать огромные каталоги айтемов аналогично тому, как языковые модели обрабатывают слова в огромных словарях.

## Отличия от PLUM

| Аспект | TIGER | PLUM |
|--------|-------|------|
| **Эмбеддинги** | Только текстовые описания айтемов | Мультимодальные эмбеддинги (аудио, видео, текст) |
| **Коллаборативный сигнал** | Не использовался напрямую в токенизации | Интегрирован в процесс токенизации через contrastive learning |
| **Охват** | Эксперименты на Amazon датасетах | Промышленное применение в YouTube |
| **Архитектура** | Более простая архитектура | Три стадии: item tokenization, continued pre-training, task-specific fine-tuning |

## Влияние на PLUM

TIGER заложил основу для развития PLUM, предоставив:
- Концепцию использования SIDs для представления айтемов
- Использование RQ-VAE для токенизации айтемов
- Основы для интеграции айтемов в языковые модели

PLUM развил идеи TIGER, добавив коллаборативный сигнал, мультимодальные представления и масштабируемость для промышленного применения.
