# Рекомендательные системы (RecSys)

## Описание

Рекомендательные системы - это класс информационных систем, которые предполагают предпочтения или интересы пользователей для рекомендации им подходящих элементов (айтемов), таких как фильмы, товары, музыка и т.д.
