# MPFormer: Adaptive Framework for Industrial Multi-Task Personalized Sequential Retriever

## Описание

MPFormer - это адаптивный фреймворк для промышленной мультизадачной персонализированной последовательной выборки (retrieval). Модель обучает кандидатогенератор, который умеет предсказывать кандидатов для различных целей (таких как лайки, клики и другие взаимодействия), при этом адаптивно распределяя бюджет между этими разными целями на персонализированной основе.

## Ключевые особенности

- **Мультизадачность**: Способность модели предсказывать кандидатов для различных типов целей (clicks, likes, etc.) в одной архитектуре
- **Персонализация**: Индивидуальный подход к каждому пользователю при распределении бюджета между разными целями
- **Адаптивность**: Система адаптивно распределяет ресурсы/бюджет между различными таргетами на основе предпочтений пользователя
- **Последовательный подход**: Модель учитывает порядок и последовательность взаимодействий пользователей

## Архитектура

MPFormer реализует следующую архитектуру:

1. **Многоцелевая сеть**: Обучается для предсказания кандидатов по разным целям (лайки, клики и т.д.)
2. **Механизм персонализированного распределения бюджета**: Адаптивно распределяет вычислительный ресурс между разными таргетами
3. **Последовательное моделирование**: Учитывает последовательность взаимодействий пользователей

## Преимущества

- Одна модель для нескольких задач, что повышает эффективность
- Персонализированное распределение ресурсов для оптимизации по разным метрикам
- Учет последовательности пользовательских действий для более точных рекомендаций
- Масштабируемость для промышленных приложений

## Применение

MPFormer особенно полезен в промышленных условиях, где необходимо оптимизировать систему сразу по нескольким метрикам (например, клики, лайки, время просмотра), и каждому пользователю требуются разные приоритеты для этих метрик.
