# Wukong: Компонент для поиска неочевидных связей между признаками

## Описание

Wukong - это специализированный компонент архитектуры GEM, состоящий из стековых факторизационных машин, который отвечает за поиск неочевидных связей между признаками из разных групп. Компонент помогает модели глубже понимать, почему пользователь ведет себя определенным образом, анализируя сложные взаимодействия между разными типами данных.

## Проблема

В рекомендательных системах часто наблюдается ситуация, когда:

- Простые корреляции между признаками легко улавливаются стандартными методами
- Сложные, неочевидные связи остаются незамеченными
- Поведение пользователей не всегда объяснимо прямыми отношениями между известными признаками
- Требуется понимание причинно-следственных связей, а не просто корреляций

## Архитектурное решение

Wukong использует следующий подход:

- **Стек факторизационных машин**: Использует несколько уровней факторизационных машин для поиска все более сложных взаимодействий
- **Поиск неочевидных связей**: Анализирует признаки с целью нахождения сложных, неинтуитивных взаимосвязей
- **Интеграция в основную архитектуру**: Результаты работы Wukong интегрируются в основной поток модели

## Преимущества

- **Обнаружение скрытых паттернов**: Возможность обнаруживать сложные зависимости, которые неочевидны из простого анализа
- **Улучшенное понимание поведения**: Помогает понимать, почему пользователь принимает определенные решения
- **Улучшение точности рекомендаций**: Учет сложных взаимодействий улучшает качество предсказаний
- **Интерпретируемость**: Позволяет лучше объяснять логику рекомендаций

## Факторизационные машины в контексте Wukong

Факторизационные машины особенно подходят для этой задачи, потому что:

- Могут моделировать парные взаимодействия между признаками
- Эффективно работают с разреженными данными
- Обобщаются на произвольные признаки (в отличие от матричной факторизации, ограниченной пользовательско-айтемной матрицей)

## Использование в GEM

Wukong является одной из трёх ключевых технических фишек архитектуры GEM, наряду с InterFormer и пирамидальной параллельной структурой для длинных последовательностей.
