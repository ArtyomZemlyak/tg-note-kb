# Допущение "замороженного мира" в LRM

## Описание

Допущение "замороженного мира" - это ключевое допущение, лежащее в основе традиционной оценки моделей с рассуждением (LRM), при котором контекст задачи статичен, а модель генерирует ответ без каких-либо прерываний. Это допущение удобно для анализа, но нереалистично для реальных сценариев использования.

## Традиционная парадигма

В рамках допущения "замороженного мира":
- Модель получает фиксированную задачу
- Генерирует полный ответ без изменений в процессе
- Среда остается неизменной на протяжении всего процесса
- Подходит для коротких задач с быстрым выполнением

## Проблемы допущения

1. **Нереалистичность для реальных сценариев**: В реальных приложениях, таких как ассистивное программирование, модели могут "думать" часы, и контекст задачи может изменяться.

2. **Переоценка точности**: Высокая точность в статических условиях сильно переоценивает реальную устойчивость моделей - производительность может падать до 60%, когда контекст меняется на поздних этапах рассуждений.

3. **Отсутствие адаптивности**: Модели не обучаются сценариям, где пользователь может вмешиваться в процесс рассуждения.

## Альтернативный подход

Новый аналитический фреймворк оценивает LRM как постоянно действующих, прерываемых агентов, вводя систематические прерывания в рассуждения модели в разные моменты времени. Это позволяет стандартизированно оценить, как тайминг влияет на производительность.

## Сценарии прерывания

1. **Прерывания с ограничением по времени**:
   - Жёсткие прерывания: Модель вынуждена немедленно прекратить рассуждения и выдать ответ
   - Мягкие прерывания: Модели даётся указание ответить быстрее

2. **Прерывания с обновлением контекста**: Изменение условий задачи в середине инференса
