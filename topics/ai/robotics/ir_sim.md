# IR-SIM: Легковесный Python-симулятор роботов

## Общее описание

**IR-SIM** (Intelligent Robot Simulator) - это открытый, легковесный симулятор роботов на основе Python, предназначенный для навигации, управления и обучения с подкреплением. Он предоставляет простую, удобную для пользователя среду с встроенным обнаружением столкновений для моделирования роботов, сенсоров и окружающей среды.

## Ключевые особенности

- **Симуляция роботов**: Поддержка различной кинематики, сенсоров и поведения
- **Простая конфигурация**: Быстрая настройка и настройка с помощью простых YAML-файлов без сложного кодирования
- **Визуализация**: Встроенная визуализация с использованием matplotlib для немедленной отладки
- **Обнаружение столкновений**: Поддержка обнаружения столкновений с настраиваемыми политиками поведения для каждого объекта
- **Мультиагентное обучение с подкреплением**: Подходит для мультиагентных/мультироботных проектов по обучению с подкреплению
- **Академический/образовательный фокус**: Идеально подходит для академического и образовательного использования с возможностями быстрого прототипирования

## Поддерживаемые возможности

### Кинематика
- Дифференциальный мобильный робот (Differential Drive)
- Омни-направленный мобильный робот (Omni-Directional)
- Робот с рулевым управлением (Ackermann Steering)

### Сенсоры
- 2D LiDAR
- Детектор поля зрения (Field of View, FOV)

### Геометрии
- Круг
- Прямоугольник
- Многоугольник
- Линия
- Бинарная сетка карты

### Поведение
- Dash (движение прямо к цели)
- RVO (движение к цели с использованием поведения Reciprocal Velocity Obstacle)

## Установка и использование

### Требования
- Python 3.9 или выше

### Установка
```bash
# Базовая установка
pip install ir-sim

# С поддержкой управления с клавиатуры
pip install ir-sim[keyboard]

# Со всеми дополнительными зависимостями
pip install ir-sim[all]

# Из исходного кода
git clone https://github.com/hanruihua/ir-sim.git
cd ir-sim
pip install -e .
```

### Пример быстрого старта
```python
import irsim

env = irsim.make('robot_world.yaml')  # Инициализация среды с файлом конфигурации

for i in range(300):  # Запуск симуляции на 300 шагов
    env.step()      # Обновление среды
    env.render()    # Визуализация среды
    if env.done(): break  # Проверка завершения симуляции
    
env.end()  # Закрытие среды
```

## Применение

- Образование и исследования в робототехнике
- Прототипирование алгоритмов для навигации и управления
- Эксперименты с обучением с подкреплением в робототехнике
- Симуляция мультиагентных систем
- Исследования планирования пути и избежания столкновений

## Демонстрации

- Несколько круглых дифференциальных роботов с избеганием столкновений RVO
- Машино-подобный робот с управлением с клавиатуры, навигирующий по бинарным картам с использованием 2D LiDAR
- Машино-подобный робот, навигирующий по сеточным картам из 3D-датасетов среды (HM3D, MatterPort3D, Gibson и т.д.)
- Роботы с возможностями обнаружения поля зрения
- Навигация с динамическими/случайными препятствиями
- Поведение ORCA с 200+ агентами с использованием реализации pyrvo

## Академические проекты, использующие IR-SIM

- **rl-rvo-nav**: RL-алгоритмы на основе RVO поведения для мультироботной навигации (RAL & ICRA2023)
- **RDA_planner**: Ускоренный планировщик движения без столкновений (RAL & IROS2023)
- **NeuPAN**: Прямая навигация точечного робота с обучением с конца в конец (T-RO 2025)
