# Scientific Machine Learning (SciML)

Scientific Machine Learning (SciML) — междисциплинарная область, объединяющая методы машинного обучения с научными вычислениями и физическими моделями. SciML использует ML-методы для решения научных и инженерных задач, часто основанных на дифференциальных уравнениях и физических законах.
