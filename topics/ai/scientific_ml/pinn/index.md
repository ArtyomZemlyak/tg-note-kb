# Physics-Informed Neural Networks (PINNs)

Physics-Informed Neural Networks (PINNs) — это класс методов машинного обучения, которые используют нейронные сети для решения задач, основанных на физических законах и дифференциальных уравнениях. PINNs интегрируют физические уравнения (обычно дифференциальные уравнения в частных производных) в архитектуру нейронной сети в виде функции потерь.
