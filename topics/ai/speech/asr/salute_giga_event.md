# Мероприятие "Салют, Гига!" - встреча с разработчиками GigaAM

## Описание мероприятия

"Салют, Гига!" - это мероприятие, где разработчики GigaAM доступны для живого общения с сообществом. Мероприятие предоставляет возможность задать вопросы по статье о GigaAM-v3, получить детали обучения мультиязычного GigaAM Max и обсудить модификации HuBERT-CTC для использования в GigaChat Audio.

## Темы обсуждения

### GigaAM-v3
- Технические детали реализации
- Распределение данных по доменам
- Эксперименты с масштабированием модели по параметрам
- Сравнение токенизаторов
- Анализ ошибок end-to-end моделей

### LLM-as-a-judge для распознавания речи
- Методы оценки качества ASR-моделей с использованием больших языковых моделей
- Side-by-Side оценка с использованием судейских моделей
- Применение Gemini 2.5 Pro для сравнения качества с Whisper-large-v3

### GigaAM Max - мультиязычная версия
- Особенности архитектуры для мультиязычной поддержки
- Модификации HuBERT-CTC для использования в GigaChat Audio

### Сравнение с конкурентами
- GigaAM-v3 ворвалась в top trending ASR-моделей на HuggingFace
- Превзошла свежий релиз OmniLingual ASR по популярности
