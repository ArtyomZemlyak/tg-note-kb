# AI Toolkit от Ostris

## Описание

AI Toolkit - это универсальный набор инструментов для обучения диффузионных моделей, разработанный Ostris - разработчиком популярного пакета для обучения диффузионных моделей. Набор инструментов разработан так, чтобы быть простым в использовании, но при этом обладать всеми возможными функциями.

## Особенности

- Обучение диффузионных моделей на потребительском оборудовании
- Поддержка как GUI, так и командной строки
- Совместимость с моделями FLUX.1-dev и FLUX.1-schnell
- Поддержка различных типов сетей, включая LoRA и LoKr
- Веб-интерфейс для управления задачами
- Возможность тренировки конкретных слоев с настраиваемыми параметрами сети
- Поддержка видеомоделей
- Поддержка SDXL и SD 1.5 в пользовательском интерфейсе

## Поддержка новых моделей

### Z-Image Turbo

AI Toolkit получил поддержку обучения LoRA для Z-Image Turbo. Поддержка реализована с помощью адаптера De-Distill, что позволяет обучать ускоренные дистиллированные модели напрямую.

#### Технические характеристики поддержки:
- Обучение возможно на 12+ GB VRAM
- Обучение персонажа на 17 изображениях с 3000 шагами занимает около 1,5 часов на RTX 5090
- Использует De-Distill адаптер для прямого обучения ускоренной модели

#### Z-Image-Turbo Training Adapter
Специальный адаптер обучения, разработанный для решения проблемы разрушения дистилляции при прямом обучении дистиллированных моделей. Адаптер замедляет процесс дедистилляции, позволяя обучать новые LoRA на дистиллированной модели без разрушения её основных характеристик. После завершения обучения адаптер удаляется, оставляя только новую информацию на дистиллированной модели, что позволяет сохранить высокую скорость генерации.

## Новые концепции и термины

- **De-Distill адаптер** - специальный метод, позволяющий обучать ускоренные дистиллированные модели напрямую, без необходимости в полной модели учителя.
- **Z-Image Turbo** - 6-миллиардная текст-в-изображение модель, разработанная Tongyi Lab (Ali Baba), использующая архитектуру Single-Stream Diffusion Transformer.
- **LoRA для диффузионных моделей** - метод параметрически эффективного обучения, адаптированный для диффузионных моделей изображений.

## Примеры применения

- Обучение персонализированных LoRA для текст-в-изображение моделей
- Адаптация моделей к конкретным стилям или объектам
- Создание специализированных моделей изображений на потребительском оборудовании
- Быстрое создание и тестирование новых адаптаций диффузионных моделей
