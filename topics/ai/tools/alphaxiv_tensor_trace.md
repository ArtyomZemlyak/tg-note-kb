# AlphaXiv Tensor Trace: 3D-визуализация трансформера

## Описание

AlphaXiv Tensor Trace - это инновационный инструмент с подробной 3D-визуализацией архитектуры трансформера, разработанный командой AlphaXiv. Инструмент позволяет пользователям взаимодействовать с трехмерной интерактивной схемой, внутри которой можно "полетать" сквозь все внутренности трансформерной модели и увидеть, как модель работает изнутри.

## Особенности

- **3D-интерактивность**: Пользователи могут перемещаться по внутренностям модели и исследовать её архитектуру в трехмерном пространстве
- **Всплывающие подсказки**: При наведении курсора на блок появляется карточка с описанием вычислений, которые за ним стоят
- **Встроенный редактор кода**: При клике на блок открывается встроенный редактор кода с точным кодом операции
- **Семантическая группировка**: Все блоки разбиты на смысловые группы: query, key, value, MLP и residual stream для удобства восприятия

## Техническая реализация

- **Базовая модель**: LLaMA 3.1
- **Библиотека**: Реализовано на основе библиотеки ggml в llama-cpp
- **Парсинг**: Использует отладочную информацию о вычислительных операциях из ggml (например, MUL_MAT, ROPE, RESHAPE, ADD)
- **Визуализация**: 3D-визуализация выполнена с помощью three.js
- **Операции**: Визуализируются ключевые операции: MUL_MAT (матричное умножение), ROPE (Rotary Positional Embedding), RESHAPE (преобразование формы), ADD (сложение)

## Применение

- **Образование**: Помогает студентам и исследователям понять внутреннюю работу трансформерных моделей
- **Исследования**: Упрощает анализ архитектурных решений и вычислительных потоков
- **Разработка**: Позволяет разработчикам моделей визуализировать и отлаживать внутренние процессы

## Преимущества

1. **Наглядность**: Позволяет визуально понимать сложные вычислительные процессы внутри трансформеров
2. **Интерактивность**: Пользователь может активно исследовать модель, а не просто пассивно смотреть
3. **Детализация**: Предоставляет подробную информацию как на высоком уровне, так и на уровне отдельных операций
4. **Образовательная ценность**: Превосходный инструмент для обучения архитектуре трансформеров
