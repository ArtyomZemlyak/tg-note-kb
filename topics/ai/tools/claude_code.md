# Claude Code - Веб-интерфейс для кодирования

## Описание

Claude Code - это веб-интерфейс для разработки программного обеспечения, основанный на AI-ассистенте Claude от Anthropic. Это бета-версия, позволяющая делегировать задачи программирования прямо из браузера без необходимости использовать терминал. Пользователи могут подключать репозитории GitHub, описывать задачи, и Claude самостоятельно выполняет все необходимые действия в облачной среде Anthropic.

## Основные возможности

- **Параллельное выполнение задач**: Запуск нескольких задач одновременно в разных репозиториях
- **Отслеживание прогресса**: Реал-тайм мониторинг выполнения задач
- **Автоматическое создание Pull Request**: Создание PR с понятными описаниями изменений
- **Анализ архитектуры**: Ответы на вопросы о структуре проектов и репозиториев
- **Интеграция с GitHub**: Прямое подключение к репозиториям GitHub для работы с кодом
- **Поддержка мобильных устройств**: Специальная версия для iOS, позволяющая кодировать с телефона

## Техническая архитектура

### Облачная среда

Claude Code работает в изолированной песочнице Anthropic с следующими характеристиками:

- **Изоляция**: Каждая задача запускается в отдельной изолированной песочнице
- **Ограничения безопасности**: Ограничения по сети и файловой системе
- **Виртуальные машины**: Выполнение задач в управляемых VM Anthropic
- **Безопасный прокси**: Все Git-взаимодействия проходят через защищенный прокси
- **Доступ к репозиториям**: Claude видит только авторизованные репозитории

### Универсальный базовый образ

Образ песочницы включает предустановленные инструменты:

- **Языки программирования**: Python, Node.js, Java, Go, Rust, C++
- **Менеджеры пакетов**: npm, yarn, pip, poetry, Maven, Gradle, cargo
- **Фреймворки тестирования и линтеры**
- **Инструменты разработки**: Git, общие инструменты сборки

### Настройка окружения

- **Сессионные хуки**: Автоматическая установка зависимостей через SessionStart
- **Настройки сети**: Три уровня доступа - ограниченный (по умолчанию), без интернета, полный интернет
- **Переменные окружения**: Поддержка через формат .env
- **Удалённая идентификация**: Переменная CLAUDE_CODE_REMOTE для различения локального и удаленного выполнения

## Безопасность и сетевая архитектура

### Меры безопасности

- **Изоляция виртуальных машин**: Каждая сессия изолирована
- **Защита учетных данных**: Конфиденциальные данные не попадают в песочницу
- **Безопасный прокси GitHub**: Обработка git-операций с ограниченными учетными данными
- **Ограниченный сетевой доступ**: По умолчанию доступ только к разрешенным доменам

### Политика сети

По умолчанию разрешены домены:
- Сервисы Anthropic (api.anthropic.com, claude.ai)
- Системы контроля версий (github.com, gitlab.com)
- Менеджеры пакетов (pypi.org, npmjs.org, rubygems.org и т.д.)
- Облачные платформы (GCP, Azure, AWS)
- Регистры контейнеров (Docker Hub, GCR и т.д.)

### Защитный прокси

- Весь исходящий трафик проходит через HTTP/HTTPS прокси
- Защита от вредоносных запросов
- Ограничение по скорости и предотвращение злоупотреблений
- Фильтрация контента

## Области применения

### Где работает лучше всего

- **Исправление багов**: Решение понятных и рутинных задач
- **Backend-изменения**: Модификации, где можно использовать тесты для проверки
- **Архитектурные вопросы**: Ответы на вопросы о структуре проекта
- **Многократная задача**: Автоматизация повторяющихся задач программирования

## Технические ограничения

- **Ограничения по скорости**: Делит лимиты с остальным использованием Claude Code
- **Параллельные задачи**: Потребляют пропорционально больше лимитов
- **Поддержка VCS**: Пока только GitHub (без GitLab или других систем контроля версий)
- **Аутентификация**: Требуется для перемещения сессий между веб и терминалом

## Миграция сессий

- **Переход с веб на терминал**: Использование кнопки "Open in CLI" для продолжения сессий локально
- **Автоматическая команда**: Генерируется команда для выполнения в терминале
- **Сохранение изменений**: Локальные изменения сохраняются во время миграции

## Новые концепции и термины

- **Claude Code**: AI-ассистент для программирования, созданный Anthropic
- **Песочница (Sandbox)**: Изолированная среда выполнения для безопасной работы с кодом
- **SessionStart хуки**: Автоматические действия, выполняемые при старте сессии
- **Secure GitHub proxy**: Безопасный прокси для обработки операций с Git
- **Web-to-terminal migration**: Возможность продолжения сессии локально из веб-интерфейса
