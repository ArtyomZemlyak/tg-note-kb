# Happy Coder - Мобильное и веб-приложение для Claude Code

## Описание

Happy Coder - это мобильное и веб-приложение для Claude Code и Codex, позволяющее использовать AI-ассистенты для программирования с любого устройства с энд-то-энд шифрованием. Проект предоставляет обертку для Claude Code с дополнительным функционалом, включая автоматическую инъекцию контекста и удаленный доступ.

## Основные возможности

- **Мобильный доступ** - Использование Claude Code с iOS и Android устройств
- **Веб-приложение** - Доступ к функциям через веб-интерфейс
- **Пуш-уведомления** - Оповещения о необходимости разрешений или ошибках Claude Code
- **Мгновенное переключение устройств** - Возможность передачи контроля между устройствами
- **Энд-то-энд шифрование** - Защита всех данных с помощью сквозного шифрования
- **Интеграция с командной строкой** - Использование как замена `claude` команды
- **Открытый исходный код** - MIT лицензия, без телеметрии

## Установка и использование

Для использования Happy Coder:

1. Установите CLI-инструмент: `npm install -g happy-coder`
2. Запускайте с помощью команды `happy` вместо `claude`
3. Используйте мобильное приложение для удаленного доступа к сессиям

## Архитектура

Happy Coder состоит из нескольких компонентов:

- **happy-cli** - Интерфейс командной строки
- **happy-server** - Сервер для зашифрованной синхронизации
- **Мобильные приложения** - iOS и Android клиенты

## Преимущества

- **Удаленный доступ** - Проверка прогресса AI-агентов без доступа к компьютеру
- **Безопасность** - Данные передаются только в зашифрованном виде
- **Удобство** - Переключение между устройствами без потери контекста
- **Автоматизация** - Возможность автоматической инъекции контекста
- **Совместимость** - Полная совместимость с существующими рабочими процессами Claude Code
