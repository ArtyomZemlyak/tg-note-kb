# CELL: Контрастные объяснения для больших языковых моделей (Contrastive Explanations for Large Language Models)

## Описание

CELL (Contrastive Explanation for Large Language Models) - это метод контрастных объяснений для анализа и понимания поведения больших языковых моделей (LLM). Основная идея заключается в создании "незначительно изменённой" версии промпта, при которой модель даёт заметно другой ответ по заданной метрике.

CELL работает по принципу: "Объяснения предполагают, что LLM выводит ответ на заданный промпт, потому что если бы промпт был немного изменён, LLM дал бы другой ответ". Это позволяет понять, почему модель выбрала конкретный ответ, анализируя, как она отреагировала бы на небольшие изменения в промпте.

## Принцип работы

1. **Генерация модифицированных промптов**: Создаются пертурбации (небольшие изменения) исходного промпта
2. **Сравнение ответов**: Записываются ответы модели на оригинальный и изменённые промпты
3. **Оценка различий**: Используется заданная метрика для оценки того, насколько различны ответы
4. **Формирование объяснений**: На основе различий формируются объяснения того, почему модель выбрала определённый ответ

## Алгоритм m-CELL

В дополнение к базовому CELL, существует модифицированная версия m-CELL (modified CELL), которая может включать:
- Более сложные стратегии изменения промпта
- Многократное сравнение с различными модификациями
- Статистический анализ стабильности ответов модели

## Применение

CELL используется для:
- Понимания влияния различных частей промпта на ответ модели
- Анализа надежности LLM в различных сценариях
- Выявления чувствительных частей входа, которые сильно влияют на вывод
- Объяснения поведения моделей в критических приложениях

## Связь с ICX360

CELL интегрирован в библиотеку [[index.md]] как один из ключевых методов контрастных объяснений. ICX360 реализует и расширяет методы CELL, предоставляя комплексный инструментарий для анализа LLM.

## Сравнение с другими методами

В отличие от:
- **Методов на основе внимания**: CELL фокусируется на изменении промпта и сравнении результатов, а не на анализе внутренних состояний модели
- **Методов на основе возмущений**: CELL использует более целенаправленный подход к сравнению, где акцент делается на значимые изменения в ответах
