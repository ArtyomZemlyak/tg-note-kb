# Обзор инструментов ИИ в базе знаний

## Краткое описание

В этой статье представлен обзор всех инструментов, фреймворков и систем ИИ, доступных в базе знаний. Инструменты классифицируются по категориям: фреймворки для работы с LLM, инструменты для вызова инструментов и планирования, фреймворки для выполнения на устройстве, а также разнообразные специализированные инструменты.

## Основные категории инструментов

### 1. Фреймворки для LLM-сервинга и инференса

#### SGLang
Высокопроизводительный фреймворк для обслуживания больших языковых моделей (LLM) и визуально-языковых моделей. Разработан для обеспечения низкой задержки и высокой пропускной способности инференса в различных конфигурациях, от одного GPU до крупных распределенных кластеров. Имеет двойную модель программирования: оптимизированный бэкенд-рантайм и интуитивный фронтенд-язык для программирования приложений с LLM.

#### SLiME
Фреймворк пост-тренировки больших языковых моделей (LLM) для масштабирования обучения с подкреплением (RL). Обеспечивает высокопроизводительную тренировку и гибкую генерацию данных для различных моделей, включая серии Qwen3, DeepSeek V3 и Llama 3.

#### DLLM (Simple Diffusion Language Modeling)
Библиотека, объединяющая обучение и оценку диффузионных языковых моделей (DLLM). Предоставляет масштабируемые пайплайны обучения и оценки с акцентом на прозрачность и воспроизводимость. Поддерживает различные архитектуры диффузионных моделей, включая BERT, LLaDA, Dream и EditFlow.

### 2. Инструменты и фреймворки для вызова инструментов и планирования

#### ReAct (Reasoning and Acting)
Подход, объединяющий рассуждение и действие в больших языковых моделях (LLM). Метод позволяет моделям генерировать рассуждения для планирования и выполнения действий, создавая цепочки "мышлений", которые помогают в решении сложных задач.

#### Toolformer
Подход, позволяющий большим языковым моделям автоматически обучаться использованию внешних инструментов. Основная идея заключается в использовании LLM для определения, когда использование инструмента может быть полезным, и включения этих знаний в процесс обучения.

#### MetaTool
Методология, облегчающая большим языковым моделям освоение инструментов с помощью аугментации мета-задач. Подход делает упор на мета-обучение для более эффективного освоения новых инструментов и систематическую аугментацию задач для лучшего обобщения.

#### ToolLLM
Система, обученная LLM освоению 16000 реальных API. Направлена на обучение больших языковых моделей использованию огромного количества реальных API как часть решения задачи масштабного обучения LLM работе с тысячами различных инструментов.

#### HuggingGPT
Система, объединяющая возможности ChatGPT с тысячами моделей и инструментов из экосистемы Hugging Face. Использует LLM для планирования, выбора подходящих моделей и оркестрации выполнения задач с использованием распределенных моделей.

#### WebGPT
Система, позволяющая языковым моделям отвечать на сложные вопросы, используя интернет-браузер для поиска и сбора информации. Модель обучается с использованием обратной связи от человека, чтобы производить точные и хорошо обоснованные ответы.

#### Барсук (от Яндекса)
Внутренний инструмент Яндекса, разработанный для автоматизации процесса создания контента с использованием больших языковых моделей (LLM). Содержит промпты для каждого этапа работы с контентом и автоматизирует рутинные задачи по подготовке промптов.

### 3. Инструменты для выполнения на устройстве

#### ExecuTorch
Фреймворк PyTorch для выполнения машинного обучения на устройстве, специально разработанный для мобильных, встраиваемых и пограничных устройств. Служит исполнительным компонентом для запуска моделей PyTorch на ресурсоограниченных устройствах.

### 4. Разнообразные специализированные инструменты

#### Claude Code
Инструмент от Anthropic для программирования с использованием ИИ.

#### Cursor
Инструмент для программирования с ИИ-помощником.

#### Qwen Code
Код-ассистент на основе модели Qwen от Alibaba Cloud.

#### Docling
Инструмент для обработки документов с использованием ИИ.

#### GGML
Фреймворк для выполнения больших языковых моделей на CPU.

#### n8n
Платформа для автоматизации рабочих процессов.

#### Chandra OCR / DeepSeek OCR
Инструменты оптического распознавания символов.
