# Yandex Alice AI LLM

## Описание

Alice AI LLM (Large Language Model) - языковая модель, разработанная Яндексом как часть экосистемы Yandex Alice AI. Модель построена на архитектуре MoE (Mixture of Experts) и используется для генерации текста и выполнения языковых задач.

## Архитектура

- **Mixture of Experts (MoE)** - архитектура, позволяющая эффективно масштабировать модели за счет динамического выбора экспертов
- Использует метод обучения с подкреплением с многокомпонентными наградами (Reinforcement Learning with multi-component rewards)
- Для улучшения и оценки качества ответов разработчики собрали корзину, состоящую из тысяч наиболее востребованных у пользователей ИИ-продуктов задач
- **Инициализация весов**: Использует инициализацию весов от Qwen3-235B для ускорения процесса обучения (speed up training)
- **YaFSDP**: Открытое решение для шардинга (sharding solution), открытое Яндексом для оптимизации распределенного обучения
- **FP8 квантование**: Используется для уменьшения кэша KV (от ~3.05GB до ~1.52GB), что значительно улучшает производительность
- **EAGLE-3 спекулятивное декодирование**: Обеспечивает 5.8x ускорение по сравнению с BF16 через спекулятивное декодирование

## Производительность

- Внутренние замеры показали, что Alice AI превзошла китайскую DeepSeek на собранной корзине задач
- В попарном слепом сравнении ответов новая модель побеждает предыдущую в 70 из 100 случаев
- Перестроили ML-архитектуру Яндекса для поддержки новой модели
- Достигнуто **14.3%** использования среди россиян по данным Mediascope (по состоянию на ноябрь, при этом DeepSeek - 9.4%)

## Интеграции

- Технологию генеративного поиска и LLM объединили в новый пайплайн
- Позволяет нейросети использовать в инференсе не только текст, но и другие объекты — изображения, карточки организаций, карты, видео
- LLM может отправлять последовательные запросы в «Поиск», чтобы собрать недостающую информацию для ответа

## Сравнение с другими моделями

Alice AI LLM показала превосходство над предыдущими моделями Яндекса и некоторыми конкурентами, включая китайскую DeepSeek, по внутренним метрикам.
