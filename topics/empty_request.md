# Пустой запрос для добавления информации

## Описание

В рамках анализа и систематизации информации в Базе Знаний, поступил запрос на добавление новой информации, но само содержимое запроса было пустым.

## Контекст

При выполнении запроса на анализ и добавление информации в Базу Знаний, был предоставлен только шаблон инструкций, но не было самой информации для добавления. Был применён трёхэтапный подход поиска:

1. Файловый поиск
2. Семантический векторный поиск
3. Уточняющий поиск по содержимому

## Структура текущей базы знаний

- **Data Science**
- **Математика (Math)**
