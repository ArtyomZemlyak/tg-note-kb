# Дискретная геометрия

## Общее описание

Дискретная геометрия - это раздел геометрии, изучающий комбинаторные свойства геометрических объектов. В отличие от классической геометрии, которая изучает непрерывные объекты, дискретная геометрия имеет дело с конечными или счетными наборами точек, многогранников, многоугольников и других геометрических структур.

## Классические проблемы

### 1. Проблема чисел поцелуев

#### Определение
Число поцелуев (kissing number) в n-мерном евклидовом пространстве - это максимальное число непересекающихся единичных шаров, которые могут касаться единичного шара.

#### Известные значения
- 1D: 2
- 2D: 6
- 3D: 12 (доказано в 1950-х годах)
- 4D: 24 (доказано в 2003 году)

### 2. Проблема Борсука

#### Формулировка
Можно ли любое тело диаметра 1 в n-мерном пространстве разбить на n+1 часть, каждая из которых имеет диаметр меньше 1?

#### Решение
- Положительный ответ для n ≤ 2
- Отрицательный ответ для n ≥ 64 (первоначально), затем для n ≥ 297
- Открытая проблема для промежуточных значений

### 3. Проблемы Кэкея и Никодима

#### Множества Кэкея
- Компактные множества минимальной площади, внутри которых можно повернуть отрезок длины 1 на 180 градусов
- Связь с гармоническим анализом

#### Множества Никодима
- Множества меры 0, содержащие отрезок единичной длины в каждом направлении
- Применение в анализе и геометрии

### 4. Проблемы на сетках

#### Подмножества сетки без равнобедренных треугольников
- Задача 6.59: определение размера наибольшего подмножества [n], не содержащего равнобедренных треугольников
- Связь с аддитивной комбинаторикой

#### Задача о тайлинге сетки
- IMO 2025, задача 6.65: минимальное число прямоугольных тайлов для покрытия сетки с одним непокрытым квадратом в каждой строке и столбце
- См. [[imo_2025_problem_6.md]]
