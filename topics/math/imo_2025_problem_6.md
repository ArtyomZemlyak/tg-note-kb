# IMO 2025, Задача 6.65: Оптимизация тайлинга на сетке

## Общее описание

Задача 6.65 из международной математической олимпиады (IMO) 2025 года касается оптимизации тайлинга (размещения тайлов) на квадратной сетке. Рассматривается квадратная сетка размером 2025×2025 (и более общего вида n×n) из единичных квадратов. Необходимо разместить прямоугольные тайлы на сетке так, чтобы каждая сторона каждого тайла лежала на линии сетки, и каждый единичный квадрат был покрыт не более чем одним тайлом. Цель - определить минимальное количество тайлов f(n), которое нужно разместить так, чтобы каждая строка и каждый столбец сетки имели ровно один единичный квадрат, не покрытый никаким тайлом.

## Формальное определение задачи

Рассмотрим квадратную сетку размера n×n, состоящую из единичных квадратов. Пусть f(n) обозначает минимальное количество прямоугольных тайлов, которое можно разместить на сетке, таких что:

1. Каждая сторона каждого тайла лежит на линии сетки
2. Каждый единичный квадрат покрыт не более чем одним тайлом
3. Каждая строка сетки содержит ровно один непокрытый единичный квадрат
4. Каждый столбец сетки содержит ровно один непокрытый единичный квадрат

## Математический контекст

### Отношение к комбинаторной оптимизации

Эта задача относится к классу задач комбинаторной оптимизации, в которой необходимо минимизировать количество используемых тайлов при заданных ограничениях. Это пример задачи оптимизации с ограничениями на покрытие.

### Связь с теорией графов

Задача может быть переформулирована в терминах теории графов:
- Рассмотрим двудольный граф, где одна доля представляет строки, а другая - столбцы
- Ребра соответствуют непокрытым ячейкам
- Условие "ровно один непокрытый квадрат в каждой строке и каждом столбце" означает, что мы должны выбрать n непокрытых ячеек, которые образуют совершенное паросочетание
- Задача сводится к оптимальному покрытию оставшихся (n² - n) ячеек минимальным числом прямоугольных тайлов

### Связь с задачами упаковки и покрытия

Эта задача связана с:
- Задачами покрытия: покрытие определенных ячеек минимальным количеством прямоугольных тайлов
- Задачами упаковки: размещение тайлов без пересечений
- Задачами планирования: оптимальное размещение с ограничениями

## Решение задачи

Найдем f(n) - минимальное количество тайлов для n×n сетки.

### Структура решения

1. Имеется n² ячеек в сетке n×n
2. Нужно оставить n ячеек непокрытыми (по одной в каждой строке и столбцу)
3. Остается (n² - n) ячеек для покрытия
4. Необходимо покрыть эти (n² - n) ячеек минимальным количеством прямоугольных тайлов

### Оптимальная стратегия

Для оптимизации числа тайлов, нужно максимизировать размер каждого тайла. Если мы можем использовать один прямоугольный тайл для покрытия большого подмножества ячеек, это будет эффективнее, чем использовать несколько маленьких тайлов.

Наиболее эффективный способ - использовать n прямоугольников, где каждый прямоугольник покрывает "остаток" от одной строки после исключения одного непокрытого квадрата.

### Конструкция решения

Один из оптимальных способов:
- Выберем непокрытые ячейки по диагонали: (1,1), (2,2), ..., (n,n)
- Для каждой строки i (i = 1, ..., n), покрываем все n-1 ячеек в строке i, кроме (i,i)
- Это можно сделать, используя n тайлов: прямоугольник для строки i покрывает (i,1), ..., (i,i-1), (i,i+1), ..., (i,n)

Таким образом, f(n) ≤ n.

### Доказательство минимальности

Можно доказать, что f(n) = n:

1. Поскольку в каждой строке есть ровно один непокрытый квадрат, в каждой строке покрыто (n-1) квадратов
2. Следовательно, в каждой строке должно быть хотя бы одно ребро тайла, разделяющее покрытые и непокрытые участки
3. Это приводит к нижней оценке f(n) ≥ n

Следовательно, f(n) = n.
