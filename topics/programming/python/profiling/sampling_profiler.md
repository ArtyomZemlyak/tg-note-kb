# Модуль profiling.sampling

## Краткое описание

Новый модуль `profiling.sampling` из PEP 799 предоставляет инструмент статистического профилирования, который позволяет анализировать производительность кода без пауз и overhead'а.

## Основные особенности

### Нулевой оверхед профилирования

- Профилирование запущенных процессов Python без влияния на производительность
- Нет необходимости в изменении кода для его профилирования
- Возможность мониторинга в реальном времени

### Высокая частота выборки

- Поддержка частоты выборки до 1,000,000 Гц
- Точное измерение производительности кода
- Поддержка потоков

### Форматы вывода

- Поддержка различных форматов вывода статистики
- Встроенные данные для создания flamegraph
- Подробная информация о вызовах функций

## Примеры использования

Модуль позволяет:

- Профилировать продакшн-приложения без остановки
- Находить узкие места в коде в реальном времени
- Анализировать производительность сложных систем

## Технические детали

PEP 799 "A dedicated profiling package for organizing Python profiling tools" устанавливает новую систему профилирования, организующую инструменты профилирования Python.
